## 🔥 头条

### 1. Anthropic 要求 Claude Code 订阅用户使用 OpenClaw 需额外付费

Anthropic 宣布自4月4日起，Claude Code 订阅用户将**不能再使用订阅额度**运行 OpenClaw 等第三方工具，需改为按量付费（pay-as-you-go），单独计费。Claude Code 负责人 Boris Cherny 称"订阅模式不适合第三方工具的使用模式"。

OpenClaw 创始人 Peter Steinberger（已于2月加入 OpenAI）表示曾尝试与 Anthropic 沟通但仅延迟了一周。"有趣的是，他们先是把一些热门功能抄进自家闭源工具，然后封锁开源。"Cherny 则回应称自己是"开源粉丝"并已提交 PR 改善 OpenClaw 的 prompt cache 效率。

> 📌 **点评**：Anthropic 此举实质是推动用户使用其官方 Claude Code IDE 而非第三方 harness。对 OpenClaw 用户意味着成本上升，但也可能加速社区向 OpenAI 阵营倾斜。

🔗 [TechCrunch 原文](https://techcrunch.com/2026/04/04/anthropic-says-claude-code-subscribers-will-need-to-pay-extra-for-openclaw-support/)

### 2. Anthropic 在私募市场备受追捧，OpenAI 需求降温

据 Rainmaker Securities 总裁 Glen Anderson 透露，**Anthropic 已成为私募二级市场最难买入的股票**——买方有约20亿美元待部署资金，却几乎找不到卖方。与此同时，约6亿美元 OpenAI 股份待售却找不到足够买家。

OpenAI 在二级市场的交易价隐含估值约7650亿美元，较其最新一轮8520亿美元估值有明显折价。Anderson 认为，Anthropic 此前与国防部的公开对抗反而"成就"了其品牌——公众将其视为"对抗大政府的英雄"，与 OpenAI 形成差异化叙事。

> 📌 **点评**：私募市场风向标意义显著。Anthropic 的"道德立场"正在转化为真金白银的市场溢价。

🔗 [TechCrunch 原文](https://techcrunch.com/2026/04/03/anthropic-is-having-a-moment-in-the-private-markets-spacex-could-spoil-the-party/)

### 4. Anthropic 4亿美元收购生物科技AI初创 Coefficient Bio

Anthropic 以约4亿美元股票交易收购了成立仅8个月的隐秘初创 **Coefficient Bio**。该公司由 Samuel Stanton 和 Nathan C. Frey 创立，二人均来自 Genentech 旗下的 Prescient Design，专注于用 AI 加速药物发现和生物研究。

约10人的团队将加入 Anthropic 的健康与生命科学部门。此举延续了 Anthropic 自去年10月推出 Claude for Life Sciences 以来的医疗布局。

> 📌 AI 公司跨界生物制药已成趋势，Anthropic 选择直接收购而非合作，显示了在生命科学领域的长期决心。

🔗 [TechCrunch 原文](https://techcrunch.com/2026/04/03/anthropic-buys-biotech-startup-coefficient-bio-in-400m-deal-reports/)

### 5. Anthropic 成立政治行动委员会 AnthroPAC

Anthropic 已向联邦选举委员会（FEC）提交文件，正式成立 **AnthroPAC**，计划在中期选举期间向两党候选人提供捐款。资金来自员工自愿捐款，上限5000美元。

此举标志着 Anthropic 在政策影响力方面进一步发力。此前已有 AI 公司在中期选举中投入约1.85亿美元。Anthropic 今年还通过 Public First Super PAC 投入至少2000万美元做广告宣传。

> 📌 AI 公司全面进军政治游说已成定局。Anthropic 与国防部的法律纠纷背景令其政治行动更具紧迫感。

### 6. AI 巨头抢建天然气发电厂：Meta、微软、Google 豪赌能源

三大科技公司近期密集宣布天然气发电项目：

- **Microsoft** → 联合 Chevron 和 Engine No. 1 在西德克萨斯建设最高 5 GW 天然气电厂
- **Google** → 与 Crusoe 合作在北德克萨斯建设 933 MW 天然气电厂
- **Meta** → Hyperion 数据中心新增7座天然气电厂，总容量达 7.46 GW（足以供应整个南达科他州）

燃气轮机供应已出现严重短缺，价格较2019年上涨195%，新订单排期至2028年，交付周期长达6年。分析指出，科技公司押注天然气这种有限资源存在风险——极端天气、价格波动和其他产业的竞争都可能成为隐患。

> 📌 AI 军备竞赛已从算力延伸到能源。7.46 GW 的 Meta Hyperion 令人震惊——这已不是数据中心，是一座城市。

🔗 [TechCrunch 原文](https://techcrunch.com/2026/04/03/ai-companies-are-building-huge-natural-gas-plants-to-power-data-centers-what-could-go-wrong/)
