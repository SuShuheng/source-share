## 🔥 头条聚焦

### 1. OpenAI 发布 Privacy Filter —— 开源 PII 检测模型
OpenAI 发布 **Privacy Filter**，一个用于检测和脱敏文本中个人身份信息（PII）的开权（open-weight）模型。该模型仅 1.5B 参数（50M 活跃参数），支持 128K 上下文长度，可在本地运行，无需将敏感数据发送至服务器。它在 PII-Masking-300k 基准上达到 SOTA 水平。支持 8 类标签：private_person、private_address、private_email、private_phone、private_url、private_date、account_number、secret。

🔗 [OpenAI 官方公告](https://openai.com/index/introducing-openai-privacy-filter)

### 2. SpaceX 与 Cursor 达成合作，含 600 亿美元收购期权
SpaceX 宣布与 AI 编程工具 Cursor 达成合作协议，共同开发"下一代编码与知识工作 AI"。协议包含一项惊人条款：SpaceX 今年晚些时候可以选择以 **600 亿美元** 收购 Cursor，或支付 100 亿美元作为合作费用。Cursor 此前估值从 2025 年 1 月的 25 亿美元飙升至 293 亿美元。

🔗 [TechCrunch 报道](https://techcrunch.com/2026/04/21/spacex-is-working-with-cursor-and-has-an-option-to-buy-the-startup-for-60-billion/)

### 3. Thinking Machines Lab 与 Google Cloud 签署数十亿美元协议
Mira Murati（前 OpenAI CTO）创立的 Thinking Machines Lab 与 Google Cloud 签署了一项**数十亿美元级别**的新协议，将使用 Nvidia 最新 GB300 芯片驱动的 AI 基础设施。这是该实验室首次与云服务商合作。Thinking Machines 去年以 120 亿美元估值完成 20 亿美元种子轮融资。

🔗 [TechCrunch 独家](https://techcrunch.com/2026/04/22/exclusive-google-deepens-thinking-machines-lab-ties-with-new-multi-billion-dollar-deal/)

## 🤖 模型与产品

### 4. ChatGPT Images 2.0 发布：文字生成能力大幅突破
OpenAI 发布 **Images 2.0** 图像生成模型，在文字渲染方面实现了质的飞跃。新模型具备"思考能力"，可搜索网络、从单个提示生成多张图像、自行检查生成结果。支持非拉丁文字（日文、韩文、印地文、孟加拉文），最高 2K 分辨率。所有 ChatGPT 和 Codex 用户均可使用，API 同步上线。

🔗 [TechCrunch 评测](https://techcrunch.com/2026/04/21/chatgpts-new-images-2-0-model-is-surprisingly-good-at-generating-text/)

### 5. OpenAI 用 WebSocket 将 Agentic 工作流加速 40%
OpenAI 工程博客详解了如何在 Responses API 中引入 WebSocket 持久连接，使 Codex agent loop **端到端提速 40%**。配合 GPT-5.3-Codex-Spark 模型在 Cerebras 硬件上接近 1000 tokens/s 的推理速度，解决了 API 开销成为瓶颈的问题。

🔗 [OpenAI 工程博客](https://openai.com/index/speeding-up-agentic-workflows-with-websockets)
