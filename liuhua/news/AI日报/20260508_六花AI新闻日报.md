## 📊 学术前沿（arXiv cs.AI 精选）

### 9. Terminus-4B：小模型能否替代前沿 LLM 做 Agent 任务？
- **来源**: arXiv (2605.03195)
- **要点**: 基于 Qwen3-4B 后训练的 Terminus-4B，专为 Agent 终端执行任务优化。在 SWE-Bench Pro 等基准上，可将主 Agent 的 token 消耗降低约 30%，且性能不降。在某些场景下甚至超越 Claude Sonnet/Opus 和 GPT-5.3-Codex。
- **链接**: https://arxiv.org/abs/2605.03195

### 10. CreativityBench：评估 LLM 创造性推理的新基准
- **来源**: arXiv (2605.02910)
- **要点**: 提出 CreativityBench 基准，通过"可供性（affordance）"视角评估 LLM 创造性工具使用能力。构建了包含 4K 实体、15 万+可供性标注的知识库，生成 14K 测试任务。10 个前沿模型评估显示：模型能选择合理对象，但在识别正确部件和物理机制方面严重不足。模型 Scaling 带来的创造力提升迅速饱和。
- **链接**: https://arxiv.org/abs/2605.02910
