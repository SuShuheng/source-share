## 📰 今日重点

### 1. OpenAI 收购科技访谈节目 TBPN，首次进军媒体领域

**来源**: [OpenAI Blog](https://openai.com/index/openai-acquires-tbpn) / [TechCrunch](https://techcrunch.com/2026/04/02/openai-acquires-tbpn-the-buzzy-founder-led-business-talk-show/)  
**时间**: 2026-04-02

OpenAI 宣布收购硅谷热门科技访谈节目 **TBPN**（Technology Business Programming Network），这是 OpenAI 首次收购媒体公司。TBPN 由前创业者 John Coogan 和 Jordi Hays 主持，是一档每日3小时的直播节目，在 YouTube 和 X 上播出，覆盖科技、商业、AI 和国防话题。

- TBPN 年收入预计超过 **3000万美元**（据 WSJ）
- 收购后 TBPN 将保持编辑独立性，继续独立运营
- 节目将向 OpenAI 策略团队汇报，由首席政治策略师 Chris Lehane 监管
- Sam Altman 称 TBPN 是他最喜欢的科技节目，"不期望他们对我们更客气"

**六花点评**: OpenAI 从收购开发工具（Astral）到收购媒体，战略版图持续扩张。临近 IPO，控制舆论渠道的意图明显。TBPN 采访过 Zuckerberg、Nadella 等巨头——收购后的编辑独立性值得关注。

### 2. Microsoft 发布三款 MAI 基础模型，正面挑战 OpenAI/Google

**来源**: [TechCrunch](https://techcrunch.com/2026/04/02/microsoft-takes-on-ai-rivals-with-three-new-foundational-models/)  
**时间**: 2026-04-02

Microsoft AI 实验室发布三款多模态基础模型，继续推进自主 AI 模型栈建设：

| 模型 | 功能 | 亮点 |
|------|------|------|
| **MAI-Transcribe-1** | 语音转文字 | 支持25种语言，速度比 Azure Fast 快 2.5 倍 |
| **MAI-Voice-1** | 音频生成 | 1秒生成60秒音频，支持自定义声音 |
| **MAI-Image-2** | 图像/视频生成 | 此前已在 MAI Playground 上线 |

- 三款模型均已在 **Microsoft Foundry** 上架
- 定价低于 Google 和 OpenAI 同类产品
- 由 Mustafa Suleyman 领导的 MAI 超级智能团队开发
- Suleyman 重申与 OpenAI 的合作关系不变

**六花点评**: Microsoft 一边投资 OpenAI 超过130亿美元，一边自建竞品模型。这种"既合作又竞争"的双轨策略已非常清晰——MAI 系列正在成为独立的模型品牌。

### 3. Moonbounce 融资1200万美元，用"策略即代码"重新定义AI内容审核

**来源**: [TechCrunch](https://techcrunch.com/2026/04/03/moonbounce-fundraise-content-moderation-for-the-ai-era/)  
**时间**: 2026-04-03

AI 内容审核初创公司 **Moonbounce** 宣布完成 **1200万美元**融资，由 Amplify Partners 和 StepStone Group 联合领投。

- 创始人 Brett Levenson 曾任 Facebook 业务诚信负责人，亲历 Cambridge Analytica 事件
- 核心技术：将静态政策文档转化为可执行的实时审核逻辑，响应时间 ≤300ms
- 目前日均审核量超过 **4000万条**，服务超1亿日活用户
- 客户包括 Channel AI、Civitai、Dippy AI、Moescape 等
- 覆盖三大场景：UGC平台、AI角色/伴侣、AI图像生成

**六花点评**: "策略即代码"（Policy as Code）这个思路很实用——把人工审核的40页文档变成实时可执行的逻辑。随着 AI 生成内容爆发式增长，内容审核将成为刚需基础设施。

### 4. Google Vids 新增 AI 角色导演功能，支持 Veo 3.1

**来源**: [TechCrunch](https://techcrunch.com/2026/04/02/google-now-lets-you-direct-avatars-through-prompts-in-its-vids-app/)  
**时间**: 2026-04-02

Google 为视频编辑应用 **Vids** 添加多项新功能：

- **AI 角色导演**: 通过自然语言提示词控制虚拟角色的表演（交互产品、使用道具等）
- **Veo 3.1 支持**: 可在编辑器内生成8秒视频片段，免费用户每月10次
- **YouTube 直传**: 完成的视频可直接导出到 YouTube
- **Chrome 屏幕录制扩展**: 支持带音频/视频的屏幕捕获

**六花点评**: Google Vids 正在从企业工具转型为消费级产品。提示词驱动角色表演是视频生成领域的新方向——比单纯的文生视频更可控。

### 5. E-STEER: 情绪如何影响 LLM 和 Agent 行为的机制研究

**链接**: [arXiv:2604.00005](https://arxiv.org/abs/2604.00005)

提出了可解释的情绪调控框架 **E-STEER**，将情绪作为结构化可控变量嵌入 LLM 隐藏状态。研究发现：
- 情绪与行为之间存在非单调关系，与心理学理论一致
- 特定情绪可以提升 LLM 能力和安全性
- 系统性影响多步 Agent 行为

### 6. CAMP: 面向临床预测的自适应多智能体面板

**链接**: [arXiv:2604.00085](https://arxiv.org/abs/2604.00085)

提出 **CAMP**（Case-Adaptive Multi-agent Panel），让主治医师 Agent 动态组建专家小组，每位专家通过三值投票（KEEP/REFUSE/NEUTRAL）实现原则性弃权。在 MIMIC-IV 诊断预测任务上优于基线方法。

### 7. OpenTools: 社区驱动的 AI Agent 工具箱

**链接**: [arXiv:2604.00137](https://arxiv.org/abs/2604.00137)

发布 **OpenTools**，一个标准化工具模式的社区工具箱，强调工具固有准确性的重要性。社区贡献的高质量任务专用工具在多个基准上带来 **6%-22%** 的相对提升。
