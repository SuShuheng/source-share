### 5. OpenAI确认macOS安全事件：Axios供应链攻击波及代码签名

OpenAI 发布官方安全公告，确认其 macOS 应用签名流程受到 **Axios npm 包供应链攻击**（3月31日事件）影响。攻击者通过被篡改的 Axios 1.14.1 版本入侵了 OpenAI 的 GitHub Actions 工作流。

- **影响范围**: ChatGPT Desktop、Codex App、Codex CLI、Atlas 的 macOS 版本
- **用户数据**: 未发现被访问或泄露的证据
- **应对措施**: 已轮换 macOS 代码签名证书，发布新版本
- **用户行动**: macOS 用户需在 **5月8日前** 更新至最新版本，旧版本届时将停止运行
- 根因：GitHub Actions 工作流使用了浮动 tag 而非固定 commit hash

> 📌 **来源**: OpenAI 官方博客 / Forbes
> 📅 **时间**: 2026-04-10 ~ 04-12
