### 1. OpenAI × Cloudflare：GPT-5.4 与 Codex 登陆 Agent Cloud
**来源**: OpenAI 官方 · 2026-04-13

Cloudflare 宣布在其 Agent Cloud 平台全面接入 OpenAI 前沿模型，包括 **GPT-5.4** 和 **Codex harness**。数百万企业客户现可直接在 Cloudflare Workers AI 边缘环境中部署 AI Agent，执行自动客服、系统更新、报告生成等生产任务。

要点：
- Codex harness 已在 Cloudflare Sandboxes 中正式可用（GA），后续也将登陆 Workers AI
- Agent Cloud 运行于 Cloudflare 全球边缘网络，保障低延迟和高可扩展性
- OpenAI 称已有超 100 万企业客户、Codex 周活 300 万、API 每分钟处理超 150 亿 token

🔗 [原文](https://openai.com/index/cloudflare-openai-agent-cloud)

### 3. 美政府鼓励华尔街银行测试 Anthropic Mythos 模型
**来源**: TechCrunch / Bloomberg · 2026-04-12

### 5. Apple 智能眼镜曝光：四种设计、2027 年上市
**来源**: TechCrunch / Bloomberg · 2026-04-12

Mark Gurman 报道 Apple 正在测试 **四种设计** 的智能眼镜，计划 2027 年发售，可能今年底公布。设计包括大/小方框和椭圆/圆形框，配色含黑色、海蓝、浅棕。功能对标 Meta Ray-Ban：拍照/录像、通话、音乐、Siri 交互——但不配备显示屏。

### 6. 全球最大轨道计算集群上线：Kepler 40 颗 GPU 在轨运行
**来源**: TechCrunch · 2026-04-13

加拿大 Kepler Communications 的在轨计算集群正式商用——10 颗卫星搭载 **40 颗 Nvidia Orin** 边缘处理器，通过激光通信互联。新客户 Sophia Space 将上传其操作系统到 6 颗 GPU 上进行在轨部署测试。

CEO Mina Mitry 强调轨道 AI 以**推理为主**而非训练，需要分布式而非集中式架构。
