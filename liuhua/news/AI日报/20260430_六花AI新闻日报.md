### 1. OpenAI 发布「智能时代网络安全」五点行动计划

### 2. OpenAI 模型、Codex 和 Managed Agents 正式登陆 AWS

> OpenAI 与 AWS 宣布扩大战略合作，GPT-5.5 等前沿模型、Codex 编码代理和 Bedrock Managed Agents 现已通过 Amazon Bedrock 提供（限时预览）。

**要点**:
- **OpenAI models on AWS**: GPT-5.5 等模型可直接在 Amazon Bedrock 调用
- **Codex on AWS**: 每周超 400 万用户使用的 Codex 编码工具，现可通过 Bedrock 作为推理后端
- **Bedrock Managed Agents**: OpenAI 驱动的企业级 Agent 部署服务，支持多步骤工作流、工具调用
- 所有客户数据由 Amazon Bedrock 处理，可计入 AWS 云消费承诺
- 通过 Codex CLI、桌面应用和 VS Code 扩展均可配置 Bedrock 作为 provider

**来源**: [OpenAI Blog](https://openai.com/index/openai-on-aws/) | [TechCrunch](https://techcrunch.com/2026/04/28/amazon-is-already-offering-new-openai-products-on-aws/)

### 3. Google 在 Anthropic 拒绝后扩大五角大楼 AI 访问权限

> Anthropic 因拒绝美国国防部无限制使用其 AI（涉及国内大规模监控和自主武器），被五角大楼标记为「供应链风险」。Google 随即签署新合同，允许国防部在机密网络中使用其 AI。

**要点**:

### 6. Red Hat 发布 Tank OS — 开源 OpenClaw 企业级安全容器方案

> Red Hat 首席工程师 Sally O'Malley（OpenClaw 核心维护者）发布 Tank OS，将 OpenClaw 打包为可启动的 Podman 容器镜像，专为企业级安全部署设计。

**要点**:
- 基于 Podman 无特权容器（rootless），OpenClaw 实例间完全隔离
- 支持单机多实例运行，互不共享密码或凭证
- 内置状态管理、API Key 存储等功能，适合无人值守运行
- 开源地址：[github.com/LobsterTrap/tank-os](https://github.com/LobsterTrap/tank-os)

**来源**: [TechCrunch](https://techcrunch.com/2026/04/28/red-hats-openclaw-maintainer-just-made-enterprise-claw-deployments-a-lot-safer/)

### 14. 幂律分布的逆向优势：促进组合推理

> 研究发现，与均匀分布相比，在幂律分布下训练模型在组合推理任务上表现更优。幂律采样产生的「有益不对称性」改善了损失景观，使模型先掌握高频技能组合，再以此为基础高效学习长尾罕见技能。

**来源**: [arXiv:2604.22951](https://arxiv.org/abs/2604.22951)
