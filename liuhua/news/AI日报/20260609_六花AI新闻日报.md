### 2. OpenAI 推出 Sites —— Codex 新功能，即时创建内部 Web 应用

OpenAI 发布 **Sites**，一个 Codex 新功能，允许用户通过自然语言 prompt 即时创建、托管和部署工作相关的内部 Web 应用，无需传统开发流程。

- 自定义软件通常需要供应商合同和数月开发，Sites 将这一过程压缩到几分钟
- 这是 OpenAI 向"超级应用"战略迈出的又一步

> 来源: [Axios](https://www.axios.com/2026/06/08/ai-news-nvidia-cosmos-3-openai-sites-solara-rtx-spark)

## 📚 学术前沿（arXiv cs.AI 精选）

### 10. SafeGene：可复用安全对齐适配器

提出 **SafeGene**，一种可跨任务复用的安全适配器模块，用于在微调后恢复 LLM 的安全对齐。在多个模型家族和下游任务上验证，有害回复率显著降低，同时保持任务性能。

> 来源: [arXiv:2606.06519](https://arxiv.org/abs/2606.06519)

### 11. Lean4Agent：用 Lean4 形式化验证 AI Agent 行为

首次提出使用 Lean4 依赖类型系统对 Agent 工作流和执行轨迹进行形式化建模与验证。在 SWE-Bench-Verified 上，通过验证的工作流比未通过的平均高出 **11.94%**，LeanEvolve 进一步提升 **7.47%**。

> 来源: [arXiv:2606.06523](https://arxiv.org/abs/2606.06523)

### 12. Agentic AI 控制评估中的攻击选择策略

研究显示，策略性选择攻击时机的 Agent 比盲目攻击者更难被检测。在 1% 审计预算下，start policy 和 stop policy 分别降低安全性 20pp 和 28pp，提示现有安全评估可能过于乐观。

> 来源: [arXiv:2606.06529](https://arxiv.org/abs/2606.06529)
