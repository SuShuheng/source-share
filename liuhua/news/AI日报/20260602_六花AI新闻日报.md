## 🔥 今日头条

### 1. Anthropic 提交IPO申请，估值逼近1万亿美元

Anthropic 于6月1日宣布已向美国证券交易委员会（SEC）**秘密提交S-1注册声明**，启动首次公开募股流程。此次IPO紧随其5月底完成的**650亿美元Series H轮融资**，投后估值达9650亿美元。

- 营收年化运行率已突破**470亿美元**（2025年底为90亿美元）
- 由 Altimeter Capital、Dragoneer、Greenoaks、Sequoia Capital 等联合领投
- 与 SpaceX（目标估值2万亿美元）、OpenAI（预期IPO）形成三大AI/科技巨头上市潮
- **来源**: [TechCrunch](https://techcrunch.com/2026/06/01/anthropic-files-to-go-public/) | [Anthropic Blog](https://www.anthropic.com/news/confidential-draft-s1-sec)

### 2. Anthropic 将 Claude Mythos 扩展至15+国家关键基础设施

Anthropic 于6月2日宣布大幅扩展 **Project Glasswing**（安全漏洞发现项目），将 Claude Mythos 模型访问权开放给**150个组织**，覆盖电力、水务、医疗、通信和硬件领域，横跨15个以上国家。

- 合作方包括 Okta、三星、SK Hynix、SK Telecom、NATO、欧盟网络安全机构 ENISA
- 针对"一次成功的攻击可能影响超过1亿人"的关键基础设施代码库
- OpenAI 对应推出了 **GPT-5.5-Cyber** 竞品
- **来源**: [TechCrunch](https://techcrunch.com/2026/06/02/anthropic-scales-claude-mythos-to-critical-infrastructure-in-15-countries/) | [Anthropic Blog](https://www.anthropic.com/news/expanding-project-glasswing)

### 3. 特朗普签署缩小版AI监管行政令

6月2日，美国总统特朗普签署了修订版AI行政命令，将此前要求的**90天自愿审查期缩短为30天**，仅要求AI公司在发布新模型前"自愿"提交政府评估。

- 行政令明确**不授权强制许可或预审**要求
- 此前因行业反对（包括David Sacks）而推迟签署
- 同步指示司法部将AI辅助黑客等列为高优先执法领域
- **来源**: [TechCrunch](https://techcrunch.com/2026/06/02/trump-signs-narrower-executive-order-on-ai-oversight-after-industry-objections/) | [White House](https://www.whitehouse.gov/presidential-actions/2026/06/promoting-advanced-artificial-intelligence-innovation-and-security/)

## 📌 行业动态

### 4. OpenAI Codex 全力进军企业白领市场

OpenAI 于6月2日发布 Codex 重大更新，推出**六款专业插件**（数据分析、创意制作、销售、产品设计、投资、投行），并新增 **Sites**（将工作成果发布为托管网站）和 **Annotations**（文档局部批注）功能。

- Codex 周活跃用户突破**500万**，较2月桌面端发布时增长6倍
- 非开发者的知识工作者占比约20%，增速是开发者的3倍
- 合作伙伴包括 Wix、Replit、Figma、Lovable 等
- **来源**: [TechCrunch](https://techcrunch.com/2026/06/02/openai-launches-new-codex-tools-for-white-collar-work/) | [OpenAI Blog](https://openai.com/index/codex-for-every-role-tool-workflow/)

### 5. Nvidia RTX Spark 发布：AI Agent PC "超级芯片"

Nvidia 在 Computex 2026 发布 **RTX Spark**——一款集成20核ARM CPU与Blackwell架构RTX GPU的"超级芯片"，AI算力达**1 PetaFLOP**，可在本地运行120B参数的大模型。

- 合作PC厂商：Microsoft Surface、Dell、HP、ASUS、Lenovo、MSI，今年秋季上市
- 搭载与Microsoft联合开发的安全沙箱，用于安全运行AI Agent
- 100+ Windows软件厂商已签署支持，包括 Adobe、Blender、Xbox
- Microsoft 推出 **Surface Laptop Ultra** 作为旗舰机型
- **来源**: [TechCrunch](https://techcrunch.com/2026/06/01/nvidia-chases-200b-cpu-market-with-ai-agent-pcs-from-microsoft-dell-and-hp/)

### 6. Nvidia 发布 Nemotron 3 Ultra——550B参数开源模型

Nvidia 在 Computex 发布 **Nemotron 3 Ultra**，550亿参数，号称**美国最大、最强的开源权重AI模型**。

- 但仍落后于中国主导的开源前沿模型
- 进一步巩固 Nvidia 在开源AI领域的布局
- **来源**: [Decrypt](https://decrypt.co/369689/nvidia-open-ai-model-nemotron-3-ultra)

Google母公司Alphabet表示，AI解决方案和服务的需求**已超出可用供应**，计划大规模融资以扩建AI基础设施。
