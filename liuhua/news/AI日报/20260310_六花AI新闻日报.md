## 📰 核心新闻摘要

### 1. OpenAI 收购网络安全初创公司 Promptfoo
- **来源：** CNBC、TechCrunch、The Next Web
- **时间：** 3月9日（美国时间）
- **要点：**
  - OpenAI 收购 AI 安全初创公司 Promptfoo
  - Promptfoo 团队将加入 OpenAI，技术将集成到 Frontier 平台
  - 用于 AI Agent 的自动化红队测试、安全测试和合规监控
  - Promptfoo 拥有超过 125,000 开发者和 30+ 家财富 500 强客户
- **影响：** OpenAI 首次直接进入 AI 应用安全领域

### 2. Anthropic 推出 Code Review AI 代码审查工具
- **来源：** TechCrunch
- **时间：** 3月9日
- **产品：** Claude Code Review
- **功能：**
  - AI 审查工具，用于捕获代码中的 bug
  - 面向 Claude for Teams 和 Claude for Enterprise 客户
  - 解决 AI 生成代码带来的 pull request 审查瓶颈
- **背景：** Claude Code 大幅提升了代码输出量，导致审查工作积压

### 3. Anthropic 起诉美国政府
- **来源：** Military.com、TechCrunch
- **时间：** 3月9日
- **事件：**
  - Anthropic 在加州联邦法院和华盛顿特区上诉法院提起两项诉讼
  - 起诉国防部将其列为"供应链风险"
  - Anthropic 拒绝允许其技术用于大规模监控和完全自主武器
  - 这是美国首次将此称号用于美国本土公司
- **影响：** 企业业务订阅今年已增长四倍

### 4. Microsoft 推出 Copilot Cowork
- **来源：** CNET、Fortune、Axios
- **时间：** 3月9日
- **合作：** 与 Anthropic 合作开发
- **功能：**
  - 基于 Anthropic 的 Claude Cowork 技术
  - 可独立完成任务：创建电子表格、运行报告、进行研究
  - 使用文件、邮件和日历信息
  - "Cowork 是新的 chat"——从监督 AI 转向完全委托任务
- **发布时间：** Agent 365 平台将于 5 月正式发布

### 5. Microsoft 推出 $99/月 E7 企业套件
- **来源：** Bloomberg、CNBC、Fortune
- **时间：** 3月9日
- **产品：** Microsoft 365 E7 套件
- **价格：** $99/月
- **包含：**
  - Word、Excel 等传统 Office 工具
  - Copilot AI 助手
  - Agent 365（AI Agent 监控管理工具）
  - Copilot Cowork（基于 Anthropic 技术）
  - Entra 身份管理工具
- **发布时间：** 5 月推出

### 6. Google 卷入 AI 音乐版权诉讼
- **来源：** Billboard
- **时间：** 3月6日提起诉讼
- **事件：**
  - 独立艺术家、词曲作者和制作人起诉 Google
  - 指控 Lyria 3 AI 模型使用 YouTube 未经授权的音乐训练
  - Lyria 3 于上个月通过 Gemini 应用推出
  - 用户可基于文本和图像提示生成 30 秒 AI 歌曲
- **核心争议：** Google 利用 YouTube 平台优势"从分发者转向竞争者"

### 7. Apple 推迟智能家居设备发布
- **来源：** Bloomberg
- **时间：** 3月9日
- **产品：** 智能家居显示器（代号 J490）
- **原因：** 等待新的 AI 和 Siri 集成开发
- **原计划：** 2025 年春季发布
- **影响：** Apple Intelligence 开发挑战影响产品路线图
