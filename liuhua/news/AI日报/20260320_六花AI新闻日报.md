### 1. OpenAI 收购 Astral，加速 Codex 生态系统

**来源**：OpenAI Official | **时间**：2026年3月19日

OpenAI 宣布将收购 Astral，将强大的开源开发者工具带入 Codex 生态系统。Astral 开发了 Python 生态中最广泛使用的开源工具，包括：
- **uv**：简化依赖和环境管理
- **Ruff**：极快的 linting 和格式化
- **ty**：跨代码库的类型安全强制

**关键数据**：
- Codex 自年初以来用户增长 3 倍，使用量增长 5 倍
- 超过 200 万周活跃用户
- Astral 的工具已被数百万 Python 开发者使用

**影响分析**：此次收购将使 Codex 能够更直接地与开发者日常依赖的工具集成，推动 AI 从单纯生成代码向参与整个开发工作流程演进。

**原文链接**：https://openai.com/index/openai-to-acquire-astral

### 2. DoorDash 推出 Tasks 应用，付钱让配送员拍摄视频训练 AI

**来源**：TechCrunch | **时间**：2026年3月19日 00:14 (北京时间)

DoorDash 推出独立的 "Tasks" 应用，允许公司向配送员支付报酬，完成旨在改进 AI 和机器人系统的任务。

**任务示例**：
- 拍摄日常任务的视频（如洗碗）
- 用其他语言录制语音
- 拍摄餐厅菜品照片展示菜单
- 拍摄酒店入口帮助配送员找到落货点

**商业模式**：
- 报酬根据工作量和复杂性预先显示
- 原始音频和视频将用于评估公司内部 AI 模型及零售、保险、酒店和技术领域合作伙伴的模型
- 覆盖美国 800 万配送员

**行业趋势**：Uber 去年底也宣布了类似计划，让司机通过上传照片等小任务赚取额外收入。

**原文链接**：https://techcrunch.com/2026/03/19/doordash-launches-a-new-tasks-app-that-pays-couriers-to-submit-videos-to-train-ai/

## 📚 学术研究精选 (arXiv cs.AI)

### 重要论文

1. **Transformers are Bayesian Networks** (arXiv:2603.17063)
   - 作者：Gregory Coppola
   - 核心发现：证明了每个 sigmoid transformer 都实现了加权循环信念传播
   - 意义：为理解 Transformer 架构提供了新的理论基础

2. **AI Scientist via Synthetic Task Scaling** (arXiv:2603.17216)
   - 作者：Ziyang Cai, Harkirat Behl
   - 核心：提出合成环境生成管道，训练能够从实践中学习的 AI 代理
   - 结果：Qwen3-4B 和 Qwen3-8B 在 MLGym 基准上分别提升 9% 和 12%

3. **Contrastive Reasoning Alignment: Reinforcement Learning from Hidden Representations** (arXiv:2603.17305)
   - 核心方法：CRAFT 框架，利用模型推理能力和隐藏表示提高对越狱攻击的鲁棒性
   - 结果：在 Qwen3-4B-Thinking 和 R1-Distill-Llama-8B 上平均提升 79.0% 的推理安全性

4. **VeriGrey: Greybox Agent Validation** (arXiv:2603.17639)
   - 核心方法：灰盒方法探索 LLM 代理的多样化行为和安全风险
   - 应用：在 Gemini CLI 和 OpenClaw 等真实案例中发现安全漏洞

5. **Facts as First Class Objects: Knowledge Objects for Persistent LLM Memory** (arXiv:2603.17781)
   - 核心发现：知识对象 (KOs) 在所有条件下达到 100% 准确率，成本降低 252 倍
   - 对比：上下文记忆在多跳推理上仅达 31.6%，KOs 达到 78.9%
