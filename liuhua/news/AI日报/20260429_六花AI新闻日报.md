## 📌 今日焦点

### 1. OpenAI 与 Microsoft 重新谈判合作伙伴关系

> OpenAI 与 Microsoft 宣布修订合作协议，为双方长期关系提供更多确定性和灵活性。

**要点**:
- Microsoft 对 OpenAI IP 的许可变为**非排他性**，有效期至 **2032年**
- OpenAI 产品将优先在 Azure 上发布，但**可在任何云服务商处提供服务**
- Microsoft 不再向 OpenAI 支付收入分成；OpenAI 向 Microsoft 的收入分成继续至 2030 年
- 此举解决了 OpenAI 此前与 Amazon 签署高达 500 亿美元交易可能引发的法律纠纷

**来源**: [OpenAI Blog](https://openai.com/index/next-phase-of-microsoft-partnership/) | [TechCrunch](https://techcrunch.com/2026/04/27/openai-ends-microsoft-legal-peril-over-its-50b-amazon-deal/)

### 2. OpenAI 开源 Codex 编排规范 Symphony

> OpenAI 发布 Symphony — 一个将项目管理看板（如 Linear）转化为 AI Agent 控制平面的开源编排规范。

**要点**:
- 每个 Open Task 自动分配 Agent 并行执行，减少人类上下文切换
- 部分团队 Pull Request 合并量增长 **500%**
- 支持任务 DAG 依赖管理，Agent 可自主创建新任务
- 基于 Codex 构建的 Agent 友好型工程工作流

**来源**: [OpenAI Blog](https://openai.com/index/open-source-codex-orchestration-symphony/) | [GitHub](https://github.com/openai/symphony)

### 6. EU 要求 Google 开放 Android AI 系统给第三方

> 欧盟委员会根据《数字市场法》(DMA)，要求 Google 在 Android 上开放 AI 功能给第三方服务。

**要点**:
- Gemini 目前享有系统级独占优势（快捷唤醒、屏幕上下文、主动建议等）
- EU 要求第三方 AI 工具也能通过热词/按键系统级调用
- 需开放屏幕上下文、本地数据访问等能力
- Google 称此举为"不当干预"

**来源**: [Ars Technica](https://arstechnica.com/ai/2026/04/europe-could-force-google-to-open-android-to-other-ai-assistants/)

## 🧪 前沿研究 (arXiv)

### 10. PExA: Text-to-SQL 新 SOTA

> PExA (Parallel Exploration Agent) 在 Spider 2.0 基准上实现 **70.2%** 执行准确率，刷新 SOTA。

- 核心思路：将 Text-to-SQL 转化为"软件测试覆盖"问题，通过并行的原子 SQL 测试用例确保语义覆盖
- 📄 [arXiv:2604.22934](https://arxiv.org/abs/2604.22934)

### 11. 幂律分布赋能组合推理

> 研究发现，在组合推理任务中，**幂律分布训练** 持续优于均匀分布训练，与直觉相悖。

- 幂律采样产生有益的不对称性，使模型先学习高频组合，再以此为基础高效学习长尾技能
- 📄 [arXiv:2604.22951](https://arxiv.org/abs/2604.22951)
