### 1. OpenAI Codex 大升级：计算机操控、浏览器集成、记忆系统、111 个插件
- **来源**: [OpenAI Blog](https://openai.com/index/codex-for-almost-everything/) / [TechCrunch](https://techcrunch.com/2026/04/16/openai-takes-aim-at-anthropic-with-beefed-up-codex-that-gives-it-more-power-over-your-desktop/)
- **时间**: 2026-04-16
- **要点**:
  - **后台计算机操控**：Codex 可在 macOS 上独立操作光标、点击和输入，多个 Agent 可并行工作，不干扰用户操作
  - **内置浏览器**：应用内嵌浏览器，用户可直接在页面上评论指导 Agent 操作，用于前端/游戏开发
  - **gpt-image-1.5 图像生成**：支持产品概念图、UI 原型、游戏素材等可视化生成
  - **记忆系统**（预览版）：Codex 能记住用户偏好、修正和之前积累的上下文，加速后续任务
  - **111 个插件集成**：新增 Atlassian Rovo (JIRA)、CircleCI、CodeRabbit、GitLab Issues、Microsoft Suite 等 90+ 插件
  - **自动化升级**：支持复用已有对话上下文、自主调度未来任务、跨天/周持续执行
  - **SSH 远程开发盒**（Alpha）、PDF/电子表格/幻灯片文件预览、PR 评论处理等
- **背景**: 此举直接对标 Anthropic 的 Claude Code 和 Cowork 桌面控制能力，OpenAI 在 AI 编码工具竞赛中加速追赶

### 2. OpenAI 发布 GPT-Rosalind：面向生命科学的前沿推理模型
- **来源**: [OpenAI Blog](https://openai.com/index/introducing-gpt-rosalind/)
- **时间**: 2026-04-16
- **要点**:
  - **GPT-Rosalind** 是 OpenAI 首个专为生命科学领域打造的前沿推理模型，覆盖药物发现、基因组学、蛋白质工程和转化医学
  - 在 BixBench（生物信息学数据分析基准）上取得已发表模型中的最佳成绩
  - 在 LABBench2 的 11 项科研任务中，6 项超越 GPT-5.4，其中 CloningQA（分子克隆方案设计）提升最为显著
  - 与 Dyno Therapeutics 合作评估：在 RNA 序列-功能预测任务中，best-of-10 提交结果超过 95% 的人类专家水平
  - 客户包括 Amgen、Moderna、Allen Institute、Thermo Fisher Scientific 等
  - 同步发布开源 **Life Sciences 研究插件**（Codex），接入 50+ 科学工具和数据库
  - 模型以 Rosalind Franklin（DNA 结构发现者）命名，通过 Trusted Access 计划提供

### 3. Anthropic 发布 Claude Design：AI 驱动的快速视觉设计工具
- **来源**: [TechCrunch](https://techcrunch.com/2026/04/17/anthropic-launches-claude-design-a-new-product-for-creating-quick-visuals/) / [Anthropic Labs](https://www.anthropic.com/news/claude-design-anthropic-labs)
- **时间**: 2026-04-17
- **要点**:
  - 用户通过自然语言描述需求，Claude 即可生成原型、幻灯片、单页文档等视觉内容
  - 支持团队设计系统导入，可读取代码库和设计文件保持品牌一致性
  - 可导出为 PDF、URL、PPTX，或直接发送至 Canva 进行二次编辑
  - Anthropic 强调与 Canva 互补而非竞争，定位面向非设计背景的创始人和产品经理
  - 由 **Claude Opus 4.7** 驱动，现已面向 Pro / Max / Team / Enterprise 用户开放研究预览
  - 发布时机特殊：Anthropic CPO Mike Krieger 因公司即将推出竞争性设计工具，已于 4 月 14 日辞去 Figma 董事会职务

## 🤖 机器人与具身智能

### 4. Physical Intelligence 发布 π0.7：通用机器人大脑的「组合泛化」突破
- **来源**: [TechCrunch Exclusive](https://techcrunch.com/2026/04/16/physical-intelligence-a-hot-robotics-startup-says-its-new-robot-brain-can-figure-out-tasks-it-was-never-taught/) / [PI Blog](https://www.pi.website/blog/pi07)
- **时间**: 2026-04-16
- **要点**:
  - π0.7 展示了 **组合泛化** 能力：能将不同场景中学到的技能重新组合，执行从未训练过的新任务
  - 最引人注目的实验：机器人在几乎没见过空气炸锅的训练数据下，通过语言指导成功操作空气炸锅烤红薯
  - 零指导时成功率约 5%，经约 30 分钟的提示词优化后升至 **95%**
  - 性能匹配此前针对单一任务训练的专家模型（咖啡制作、衣物折叠、纸箱组装等）
  - 研究人员坦言标准化机器人评估基准缺失，结果主要与自身此前模型对比
  - Physical Intelligence 已融资超 **$10 亿**，估值 **$56 亿**
