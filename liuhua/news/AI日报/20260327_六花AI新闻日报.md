#### 2. **Cohere 发布开源语音转文字模型 Transcribe**
- **发布方**：Cohere
- **时间**：2026年3月26日
- **要点**：
  - 20亿参数轻量级模型，支持消费级 GPU 自托管
  - 支持14种语言（含中英日韩）
  - Hugging Face Open ASR 榜单平均 WER 5.42，超越竞品
  - 人工评估平均胜率 61%
  - 处理速度：525分钟音频/分钟
  - API 免费开放，集成到 North 企业平台

#### 3. **Mistral 发布开源文字转语音模型 Voxtral TTS**
- **发布方**：Mistral AI
- **时间**：2026年3月26日
- **要点**：
  - 支持9种语言（英法德西荷葡意印地阿拉伯）
  - 基于 Ministral 3B，可部署在智能手表/手机/笔记本等边缘设备
  - TTFA（首音频延迟）仅 90ms
  - RTF 6x（10秒音频约1.6秒生成）
  - <5秒样本即可克隆声音特征（口音、语调、语速）
  - 与 ElevenLabs、Deepgram、OpenAI 直接竞争
- **来源**：[TechCrunch](https://techcrunch.com/2026/03/26/mistral-releases-a-new-open-source-model-for-speech-generation/)
- **核验状态**：✅ 已核验
