### 1. OpenAI CEO Sam Altman 住所遭燃烧弹袭击，嫌疑人已被捕

4月11日凌晨约4时，一名20岁男性嫌疑人向旧金山North Beach的Altman住所投掷燃烧弹（Molotov cocktail），导致外围大门起火。约一小时后，该嫌疑人又在OpenAI旧金山总部外威胁纵火，被警方当场逮捕。

Altman随后在个人博客发文回应，称"低估了言语和叙事的力量"，呼吁AI行业降级论调与手段，并表示理解反技术情绪，但相信技术进步能让未来更美好。

## 📰 行业动态

### 3. Anthropic 临时封禁 OpenClaw 创始人 Claude 账号

OpenClaw 创始人 Peter Steinberger（现为 OpenAI 员工）周五凌晨发文称 Anthropic 以"可疑活动"为由暂停其账号。数小时后帖子疯传，账号恢复。此前 Anthropic 上周刚调整定价，要求 OpenClaw 用户通过 API 单独付费使用 Claude（不再包含在订阅内）。

Steinberger 表示自己只是在遵守新规则使用API测试，以确保 OpenClaw 对 Claude 的兼容性。他还透露："一方欢迎了我，另一方发了律师函。"

**来源**: TechCrunch ✅（原文核验通过）

### 4. ChatGPT 推出 $100/月 Pro 计划

OpenAI 正式发布 $100/月 Pro 计划，填补了 $20 Plus 与 $200 Pro 之间的价格空白。新计划提供比 Plus 高5倍的 Codex 编码额度。OpenAI 表示目前全球每周有超过300万人使用 Codex，过去三个月增长5倍。

值得注意的是，$100 计划在5月31日前提供更高的 Codex 额度，暗示这是与 Anthropic $100/月 Claude 方案的直接竞争。

**来源**: TechCrunch ✅（原文核验通过）

### 5. Anthropic 限制 Mythos 模型发布引发争议

Anthropic 本周表示因 Mythos 模型过于擅长发现软件安全漏洞，将限制其发布范围，仅与 AWS、JPMorgan Chase 等大型企业和关键基础设施运营商共享。

但质疑声四起：AI 安全创业公司 Aisle 声称用更小的开源模型复现了 Mythos 大部分能力；业界观察者指出，这种"企业专属"策略也可能是一种阻止模型蒸馏、锁定大企业合同的商业模式。

**来源**: TechCrunch ✅（原文核验通过）

## 🔒 安全与合规

### 6. OpenAI 回应 Axios 开发者工具供应链攻击

3月31日 Axios npm 包被入侵后，OpenAI 确认其 macOS 应用签名流程受到影响——恶意版本的 Axios（1.14.1）在 GitHub Actions 工作流中执行，该工作流可访问代码签名证书。

OpenAI 表示：
- 无证据表明用户数据、系统或知识产权被泄露
- 已轮换 macOS 代码签名证书并发布新版应用
- 5月8日起旧版 macOS 应用将停止工作
- 根本原因是 GitHub Actions 使用浮动 tag 而非固定 commit hash

**来源**: OpenAI 官方安全公告 ✅（原文核验通过）
