## 🔥 今日爆火项目

### 1. **Claude Code** - 终端AI编码助手
- **项目**: [anthropics/claude-code](https://github.com/anthropics/claude-code)
- **⭐ 今日增长**: 2,496 stars
- **核心用途**: Claude Code是一个在终端中运行的智能编码工具，能够理解代码库，通过自然语言命令执行日常任务、解释复杂代码并处理git工作流。
- **为何值得关注**: 
  - 🔥 最新推出的AI编码工具，由Anthropic开发
  - 🔥 支持在终端、IDE中直接使用，或通过GitHub @claude 调用
  - 🔥 提供完整的插件生态系统，扩展功能强大
  - 🔥 支持自定义代理和技能配置

### 2. **VibeVoice** - 微软开放语音AI
- **项目**: [microsoft/VibeVoice](https://github.com/microsoft/VibeVoice)
- **核心用途**: 微软开源的前沿语音AI模型系列，包含文本转语音(TTS)和自动语音识别(ASR)模型。
- **为何值得关注**:
  - 🚀 **60分钟单次处理**: VibeVoice-ASR能一次性处理长达60分钟的音频，生成包含说话人、时间戳和内容的结构化转录
  - 🌐 **多语言支持**: 原生支持50多种语言
  - ⚡ **vLLM推理支持**: 提供更快的推理速度
  - 🔥 **已被Hugging Face Transformers集成**: 可直接通过Transformers库使用

### 3. **Oh My CodeX (OmX)** - Claude Code工作流增强
- **项目**: [Yeachan-Heo/oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex)
- **⭐ 今日增长**: 2,852 stars
- **核心用途**: OpenAI Codex CLI的工作流层，提供更好的提示、工作流和运行时支持。
- **为何值得关注**:
  - 🔥 为Claude Code提供结构化工作流
  - 🔥 提供关键技能如 `$deep-interview`、`$ralplan`、`$team`、`$ralph`
  - 🔥 支持并行执行和持久化完成循环
  - 🔥 完整的项目状态管理和跟踪

## 🚀 技术新星

### 4. **TimesFM** - Google时间序列基础模型
- **项目**: [google-research/timesfm](https://github.com/google-research/timesfm)
- **核心用途**: Google研究院开发的时间序列预测基础模型，支持长达16k上下文长度。
- **为何值得关注**:
  - 📊 最新版本TimesFM 2.5，参数量从500M减少到200M
  - 🔥 支持连续分位数预测和自定义热点词
  - 🔥 已集成到BigQuery作为官方产品
  - 🔥 适用于金融、气象等时间序列预测场景
