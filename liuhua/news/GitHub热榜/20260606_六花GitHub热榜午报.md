## 🔥 今日爆款项目

### 1. chopratejas/headroom — LLM 上下文压缩层
- **Stars**: ⭐ 14,255（今日 +2,503）| **Forks**: 900
- **语言**: Python
- **链接**: https://github.com/chopratejas/headroom
- **核心用途**: 在 AI Agent 与 LLM 之间插入压缩层，对工具输出、日志、RAG 片段、文件内容进行压缩后再送入模型，可减少 60-95% 的 token 消耗，同时保持回答质量不变。
- **为何值得关注**: 支持 Library / Proxy / MCP Server 三种接入模式，兼容 Claude Code、Codex、Cursor 等主流 Agent；内置 6 种压缩算法（JSON SmartCrusher、AST CodeCompressor、HuggingFace Kompress-base），且压缩可逆（CCR）——原始内容从未删除。在真实 Agent 工作负载上实测：代码搜索 token 减 92%、SRE 事件调试减 92%。今日暴涨 2500+ star，属于现象级项目。

### 2. lfnovo/open-notebook — 开源 NotebookLM 替代品
- **Stars**: ⭐ 25,786（今日 +1,142）| **Forks**: 2,970
- **语言**: TypeScript
- **链接**: https://github.com/lfnovo/open-notebook
- **核心用途**: Google NotebookLM 的开源替代品，支持 PDF、视频、音频、网页等多模态内容整理，具备全文/向量搜索、AI 对话、多播客生成（1-4 位自定义发言人）等能力。
- **为何值得关注**: 支持 18+ AI Provider（OpenAI、Anthropic、Ollama、LM Studio 等），完全自托管、数据本地化。相比 NotebookLM 的 2 人播客限制，可生成多达 4 位自定义角色的播客。Docker 一键部署，适合个人知识管理。

### 3. NVIDIA/cosmos — 物理世界模型平台
- **Stars**: ⭐ 9,339（今日 +494）| **Forks**: 595
- **语言**: Jupyter Notebook
- **链接**: https://github.com/NVIDIA/cosmos
- **核心用途**: NVIDIA 推出的物理 AI 开放平台，包含世界模型、数据集和工具链，用于机器人、自动驾驶、智慧基础设施等领域。
- **为何值得关注**: 最新 Cosmos 3 采用统一 MoE（Mixture-of-Transformers）架构，同时处理语言、图像、视频、音频和动作序列。分为 Reasoner（世界理解/推理）和 Generator（世界生成/模拟）两大运行面。这是 NVIDIA 在具身智能赛道的核心开源项目。

### 4. NousResearch/hermes-agent — 自我进化的 AI Agent
- **语言**: Python
- **链接**: https://github.com/NousResearch/hermes-agent
- **核心用途**: Nous Research 出品的自学习 AI Agent，内置闭环学习循环——从经验中自动创建技能、使用中自我改进、自动将知识持久化，跨会话构建用户模型。
- **为何值得关注**: 支持 Telegram/Discord/Slack/WhatsApp/Signal 全平台接入，可在 $5 VPS 或 GPU 集群上运行；支持 200+ 模型切换（OpenRouter）、定时自动化、子 Agent 并行。其"自学习循环"设计理念独树一帜，是 Agent 持久化记忆方向的重要实践。

### 5. MemPalace/mempalace — 开源 AI 记忆系统
- **语言**: Python
- **链接**: https://github.com/MemPalace/mempalace
- **核心用途**: 本地优先的 AI 记忆系统，以逐字存储（verbatim）方式保存对话历史，通过语义搜索检索。将记忆组织为"宫殿"结构——人物和项目是"翼楼"，话题是"房间"，原始内容是"抽屉"。
- **为何值得关注**: LongMemEval 基准测试 R@5 达 96.6%（纯语义搜索，零 API 调用），混合模式达 98.4%。后端可插拔（默认 ChromaDB），完全本地运行，适合为 Claude Code / Gemini CLI 等 Agent 注入长期记忆。
