## 🔥 今日热榜精选

### 1. Claude HUD - Claude Code 实时监控插件
**项目**: [jarrodwatts/claude-hud](https://github.com/jarrodwatts/claude-hud)  
**核心用途**: Claude Code 插件，实时显示上下文使用率、活动工具、运行代理和待办进度  
**为何值得关注**: 
- 🎯 **真眼洞察**: 在 Claude Code 底部实时显示会话状态，无需切换窗口
- 📊 **原生数据**: 直接从 Claude Code 获取 token 数据，支持 1M 上下文会话
- ⚡ **轻量级**: 使用原生 statusLine API，无需 tmux 或额外窗口
- 🎨 **高度可定制**: 支持 Full/Essential/Minimal 三种预设，可调整颜色和阈值

**安装方式**:
```bash
/plugin marketplace add jarrodwatts/claude-hud
/plugin install claude-hud
/claude-hud:setup
```

### 4. Newton - GPU 加速物理模拟引擎
**项目**: [newton-physics/newton](https://github.com/newton-physics/newton)  
**核心用途**: 基于 NVIDIA Warp 的 GPU 加速物理模拟引擎，面向机器人学家和仿真研究者  
**为何值得关注**:
- 🚀 **GPU 加速**: 充分利用 GPU 进行大规模并行计算
- 🤖 **机器人专用**: 集成 MuJoCo Warp，支持机器人仿真
- 🎨 **OpenUSD 支持**: 原生支持 OpenUSD 格式
- 🔬 **可微分**: 支持可微分仿真，适合研究和优化
- 🏢 **强大背景**: 由 Disney Research, Google DeepMind, NVIDIA 联合发起

**示例场景**:
- 基础物理（钟摆、URDF、关节）
- 机器人（CartPole, H1, AnyMal）
- 布料仿真（弯曲、悬挂、滚动）
- 逆运动学（Franka, H1）
- 软体和多物理场

**安装方式**:
```bash
pip install "newton[examples]"
python -m newton.examples basic_pendulum
```
