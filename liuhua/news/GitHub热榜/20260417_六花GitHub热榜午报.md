## 🔥 今日焦点

### 1. forrestchang/andrej-karpathy-skills
- **链接**: https://github.com/forrestchang/andrej-karpathy-skills
- **核心用途**: 一个 CLAUDE.md 文件，基于 Andrej Karpathy 对 LLM 编码陷阱的观察，提炼出四大原则来改善 Claude Code 行为。
- **为何值得关注**: Karpathy 近期在社交媒体上系统总结了 LLM 编程的三大问题——错误假设与不寻求澄清、过度工程化、随意修改无关代码。本项目将其转化为可操作的四个原则（Think Before Coding / Simplicity First / Surgical Changes / Goal-Driven Execution），并已上线 Claude Code 插件市场。对使用 AI 编码工具的开发者有直接实用价值。

### 2. thedotmack/claude-mem
- **链接**: https://github.com/thedotmack/claude-mem
- **核心用途**: Claude Code 持久化记忆压缩系统——自动捕获编码会话中的所有操作，用 AI 压缩语义摘要，并在后续会话中注入相关上下文。
- **为何值得关注**: 解决了 AI 编码助手"每次会话从零开始"的核心痛点。支持 Claude Code / Gemini CLI / OpenCode 多平台，提供 10+ 搜索端点、Web UI 查看器、Chroma 向量库混合检索，还集成了 OpenClaw Gateway。已迭代至 v6.5.0，社区活跃度极高。

### 3. vercel-labs/open-agents
- **链接**: https://github.com/vercel-labs/open-agents
- **核心用途**: Vercel 官方开源的后台编码 Agent 参考应用——包含 Web UI、Agent 运行时、沙箱编排和 GitHub 集成，从提示到代码变更全流程。
- **为何值得关注**: 三层架构设计清晰（Web → Agent Workflow → Sandbox VM），Agent 在沙箱外运行、通过工具操作沙箱，实现了执行与隔离的解耦。基于 Vercel Workflow SDK 实现持久化多步骤执行，支持沙箱快照恢复、自动提交/PR。这是目前最完整的"自建云端编码 Agent"开源模板。

### 4. lsdefine/GenericAgent
- **链接**: https://github.com/lsdefine/GenericAgent
- **核心用途**: 极简自进化 Agent 框架——仅约 3K 行核心代码，通过 9 个原子工具赋予 LLM 系统级控制能力（浏览器、终端、文件系统、键鼠、屏幕视觉、ADB），并自动将任务执行路径"结晶"为可复用技能。
- **为何值得关注**: "不预加载技能，而是让技能自己生长"的设计理念非常独特。核心 Agent Loop 仅约 100 行，上下文窗口 <30K（远低于其他 Agent 的 200K-1M），且整个仓库从安装 Git 到每个 commit 都由 Agent 自主完成，是一个引人注目的自举（self-bootstrap）案例。

### 5. obra/superpowers
- **链接**: https://github.com/obra/superpowers
- **核心用途**: 编码 Agent 的完整软件开发工作流框架——包含头脑风暴、Git Worktree 管理、计划编写、子代理驱动开发、TDD、代码审查、分支收尾等可组合"技能"。
- **为何值得关注**: 覆盖了从需求讨论到代码合并的全生命周期，强调真正的红绿 TDD 和 YAGNI/DRY 原则。支持 Claude Code、Cursor、Codex、OpenCode、Copilot、Gemini 等多平台插件市场安装。子代理驱动开发模式可以让 AI 自主工作数小时不偏离计划。

## 🤖 AI & Agent 热门

### 6. virattt/ai-hedge-fund
- **链接**: https://github.com/virattt/ai-hedge-fund
- **核心用途**: AI 对冲基金概念验证——部署多个模拟知名投资人的 Agent（巴菲特、达利欧、伍德、塔勒布等），加上估值、情绪、基本面、技术面分析 Agent，协同做出交易决策。
- **为何值得关注**: 将多 Agent 协作范式应用到金融投资领域，每个"投资人 Agent"有独立的投资哲学和决策逻辑。支持 CLI 和 Web 应用两种运行方式，还内置回测器。教育目的的项目但架构设计精巧。

### 7. Donchitos/Claude-Code-Game-Studios
- **链接**: https://github.com/Donchitos/Claude-Code-Game-Studios
- **核心用途**: 将单个 Claude Code 会话变成完整的游戏开发工作室——49 个专业 Agent、72 个工作流技能、12 个自动化 Hook、39 个文档模板，模拟真实工作室层级。
- **为何值得关注**: 三层 Agent 架构（Director / Department Lead / Specialist），每个 Agent 有明确职责、升级路径和质量门控。从创意头脑风暴到发布的全流程覆盖，是将多 Agent 协作应用到创意产业的有趣实践。

### 8. Lordog/dive-into-llms
- **链接**: https://github.com/Lordog/dive-into-llms
- **核心用途**: 《动手学大模型》系列编程实践教程——由上海交通大学课程讲义拓展而来，涵盖微调部署、提示学习、知识编辑、数学推理、模型水印、越狱攻击、多模态、GUI Agent、RLHF 安全对齐等主题。
- **为何值得关注**: 国内最系统的大模型入门编程教程之一，由学术团队维护，公益免费。近期新增了数学推理、GUI Agent、隐写术等前沿主题，并联合华为昇腾推出了《大模型开发全流程》系列课程。

## 🛠️ 工具与开源

### 9. jamiepine/voicebox
- **链接**: https://github.com/jamiepine/voicebox
- **核心用途**: 开源本地语音合成工作室——ElevenLabs 的免费替代品。支持 5 个 TTS 引擎（Qwen3-TTS、LuxTTS、Chatterbox 等）、23 种语言、声音克隆、后期处理效果和多音轨时间线编辑器。
- **为何值得关注**: 本地优先、完全隐私、Tauri (Rust) 构建。5 个引擎各有特长，从轻量级英文 TTS 到多语言高质量克隆。REST API 优先设计方便集成。跨平台支持 macOS (MLX/Metal)、Windows (CUDA)、Linux。

### 10. google/magika
- **链接**: https://github.com/google/magika
- **核心用途**: Google 出品的 AI 驱动文件类型检测工具——基于深度学习，在 ~100M 样本上训练，覆盖 200+ 内容类型，平均准确率约 99%，单 CPU 推理约 5ms/文件。
- **为何值得关注**: 已在 Google 内部大规模使用（每周处理数千亿文件），用于 Gmail、Drive、Safe Browsing 的安全扫描路由。提供 Rust CLI、Python API、JS/TS npm 包。学术成果发表于 ICSE 2025。实用工具级别的项目。

### 11. pascalorg/editor
- **链接**: https://github.com/pascalorg/editor
- **核心用途**: 基于 React Three Fiber + WebGPU 的 3D 建筑编辑器——创建和分享 3D 建筑项目，支持墙体、楼板、天花板、屋顶等建筑元素的实时编辑。
- **为何值得关注**: 技术栈现代（React 19 + Next.js 16 + Three.js WebGPU + Zustand），采用 Dirty Node 模式实现高效几何更新，ECS 风格的 System 架构。Turborepo monorepo 结构清晰。对 Web 3D 编辑器架构有很好的参考价值。
