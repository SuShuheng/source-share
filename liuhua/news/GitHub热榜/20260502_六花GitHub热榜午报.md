## 🔥 今日之星（今日新增 Star 最高）

### 1. mattpocock/skills
- **链接**：https://github.com/mattpocock/skills
- **今日新增**：⭐ +3,649 | 总 ⭐ 51,893
- **核心用途**：为 Claude Code / Codex / Copilot 等编程 Agent 提供"真实工程技能"的技能集（Skills）。直接来自作者日常使用的 `.claude` 目录。
- **为何值得关注**：这不是"氛围编程"（vibe coding）的产物。Matt Pocock（TypeScript 社区知名教育家）将自己数十年的软件工程经验提炼为可复用的 Agent 技能——包括需求对齐（/grill-me）、TDD、架构改善、系统调试等。核心理念是：AI 编程的瓶颈不在代码生成，而在**工程方法论**。在 AI Agent Skill 生态中，这是目前最具影响力的项目之一。

### 2. warpdotdev/warp 🆕 开源
- **链接**：https://github.com/warpdotdev/warp
- **语言**：Rust | **协议**：AGPL v3 + MIT
- **核心用途**：终端原生（terminal-native）的 Agent 开发环境。内置编程 Agent，也支持接入 Claude Code、Codex、Gemini CLI 等外部 Agent。
- **为何值得关注**：Warp 终端**刚刚开源**，OpenAI 是创始赞助商。其 Agent 工作流由 GPT 模型驱动，并且已部署数千个 "Oz Agent" 进行 issue 分类、PR review 等自动化协作。可通过 [build.warp.dev](https://build.warp.dev) 实时观看 Agent 工作过程。这是终端工具 Agent 化的标志性事件。

### 3. 1jehuang/jcode
- **链接**：https://github.com/1jehuang/jcode
- **语言**：Rust | **今日新增**：⭐ +404
- **核心用途**：新一代 Coding Agent Harness（编程 Agent 运行框架），主打多会话工作流、极限性能和语义记忆系统。
- **为何值得关注**：性能数据惊人——单会话内存仅 27.8 MB（Claude Code 386.6 MB 的 1/14），首帧渲染 14ms（Claude Code 3.4s 的 1/245）。内置向量记忆系统可实现类人类的"自动回忆"。这是对当前 Coding Agent 性能瓶颈的一次根本性挑战。

## 🤖 AI Agent 生态项目

### 4. TauricResearch/TradingAgents
- **链接**：https://github.com/TauricResearch/TradingAgents
- **语言**：Python | **协议**：MIT
- **核心用途**：多 Agent LLM 金融交易框架。模拟真实交易公司的运作模式——基本面分析师、情绪分析师、技术分析师、研究员团队、交易员、风控和投资组合经理，全部由 LLM Agent 扮演。
- **为何值得关注**：v0.2.4 刚发布，支持 GPT-5.x、Gemini 3.x、Claude 4.x、DeepSeek、Qwen、GLM 等多家模型。支持 Docker 部署，带交互式 CLI。是"多 Agent 协作"范式在金融领域的标杆实现。

### 5. obra/superpowers
- **链接**：https://github.com/obra/superpowers
- **核心用途**：完整的编程 Agent 软件开发方法论框架。从需求梳理 → 设计文档 → 实现计划 → 子 Agent 驱动开发 → TDD → Code Review → 分支合并，全流程自动化。
- **为何值得关注**：已进入 Anthropic 官方 Claude 插件市场，同时支持 Codex、Cursor、OpenCode、Copilot、Gemini 等主流平台。核心理念：**Agent 的技能应该是强制流程，而非建议**。代表了一种将软件工程最佳实践"硬化"为 Agent 行为的思路。

### 6. lukilubs/craft-agents-oss
- **链接**：https://github.com/lukilabs/craft-agents-oss
- **语言**：TypeScript | **协议**：Apache 2.0
- **核心用途**：Agent 原生（Agent Native）桌面应用。使用 Claude Agent SDK + Pi SDK，支持多 LLM 提供商、MCP 服务器、REST API 自动发现与连接。具备会话管理、自动化事件驱动、文件 Diff 查看器等功能。
- **为何值得关注**：由 Craft（知名笔记应用）团队开源。其"对话式接入 API"的体验令人印象深刻——只需告诉 Agent "添加 Linear 作为数据源"，它会自动查找公开 API、阅读文档、配置凭证。代表了 Agent 交互从 CLI 向富 GUI 的演进方向。

### 7. browserbase/skills
- **链接**：https://github.com/browserbase/skills
- **语言**：JavaScript | **今日新增**：⭐ +334
- **核心用途**：为 Claude Code 提供浏览器自动化能力的技能集。支持远程 Browserbase 会话、反检测、验证码破解、住宅代理等。
- **为何值得关注**：让编程 Agent 获得"看网页、操作网页"的能力。包括 UI 测试、Cookie 同步、网页抓取、搜索等 10+ 个技能模块。是 Agent Skill 生态的重要补充。

### 8. simstudioai/sim
- **链接**：https://github.com/simstudioai/sim
- **语言**：TypeScript | **总 ⭐** 28,081
- **核心用途**：AI Agent 的构建、部署和编排平台。定位为"AI 劳动力的中央智能层"。
- **为何值得关注**：Star 数已达 2.8 万，说明社区认可度高。作为 Agent 编排层，填补了"单个 Agent"到"Agent 团队协作"之间的空白。
