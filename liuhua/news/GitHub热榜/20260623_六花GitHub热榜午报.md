## 🔥 AI 视频生成与制作

### 1. palmier-io/palmier-pro ⭐ [Trending #1](https://github.com/palmier-io/palmier-pro)
**核心用途**: macOS AI 原生视频编辑器

**亮点**:
- Swift 从零构建的视频编辑器，以 Premiere Pro 为目标
- 内置 SOTA 生成式 AI（Seedance、Kling、Nano Banana Pro）
- 通过 MCP 与 Claude/Codex/Cursor 集成
- 开源编辑器 + MCP 服务器（生成式 AI 功能需订阅）
- 仅支持 macOS 26 (Tahoe) Apple Silicon

**为何值得关注**:
真正将 AI 深度融入视频工作流的桌面工具，降低专业视频制作的 AI 难度，适合创作者和小团队。

---

### 2. calesthio/OpenMontage ⭐ [Trending #2](https://github.com/calesthio/OpenMontage)
**核心用途**: 世界首个开源智能视频制作系统

**亮点**:
- 12 条生产管线、52 个工具、500+ Agent 技能
- 支持真实视频（非静态图转视频）制作，使用免费素材
- Agent-first 架构：你的 AI 编程助手就是编排者
- 生成式 AI + 实时网页研究 + 完整质量检查
- 支持免费本地模式（Piper TTS + Archive.org + Remotion/FFmpeg）
- 从参考视频生成生产计划，或自然语言描述直接生成视频

**为何值得关注**:
Agent-first 视频制作管道的标杆，可在不付费 API 的情况下生成高质量视频，开源社区可扩展性强。

## 🤖 RAG 与大模型

### 3. infiniflow/ragflow ⭐ [24h +105](https://github.com/infiniflow/ragflow)
**核心用途**: 开源 RAG 引擎 + Agent 能力

**亮点**:
- 融合 RAG 与 Agent，创建大模型的优质上下文层
- 支持 DeepSeek v4、GPT-5 等最新模型
- 多模态文档理解（PDF/DOCX 图片）
- 丰富的模板与可视化 Chunking
- 支持多渠道对话（飞书、Discord、Telegram、Line 等）
- 与 OpenClaw 有官方 Skill 集成

**为何值得关注**:
RAG 领域的领先开源方案，Agent 能力增强，适合企业级 AI 应用构建，国产开源项目走向国际化。

## ☸️ 云原生与基础设施

### 4. meshery/meshery ⭐ [24h +196](https://github.com/meshery/meshery)
**核心用途**: 云原生多集群管理平台

**亮点**:
- CNCF 项目，380+ 云原生集成
- 单一平台管理多集群、多云、混合云
- GitOps 可视化配置，无需手写 YAML
- 支持 380+ 集成，可扩展插件生态

**为何值得关注**:
云原生基础设施编排的现代化工具，降低 K8s 管理复杂度，适合 DevOps 和平台工程团队。

## 📊 可观测性与监控

### 5. openobserve/openobserve ⭐ [Trending 热点](https://github.com/openobserve/openobserve)
**核心用途**: 开源可观测性平台（日志、指标、链路、RUM）

**亮点**:
- 140x 存储成本优势（Parquet 列式存储 + S3）
- 单二进制部署，2 分钟启动
- OpenTelemetry 原生，无供应商锁定
- SQL + PromQL 查询，统一平台
- Rust 构建，高性能、内存安全
- 支持 RUM（真实用户监控）、会话回放
- 多租户、Stateless 架构、HA 集群
- 免费层 50GB/天

**为何值得关注**:
Datadog/Splunk 的 140x 成本优势开源替代，适合成本敏感团队，单一平台替代多种监控工具。

## 💡 总结

- **视频生成领域**: OpenMontage 和 Palmier Pro 是 Agent-first 视频制作的代表，OpenMontage 功能更全面，Palmier Pro 专注 macOS 体验。
