## 🔥 今日爆火项目 TOP 10

### 1. colbymchenry/codegraph ⭐ 18,829 (+2,434/天)
- **链接**: https://github.com/colbymchenry/codegraph
- **核心用途**: 为 AI 编程助手（Claude Code、Codex、Cursor、OpenCode、Hermes Agent）提供预索引代码知识图谱，减少 Token 消耗约 35%、工具调用减少约 70%，完全本地运行。
- **为何值得关注**: AI 编程助手的"外挂大脑"。通过 SQLite 本地索引代码的符号关系、调用图和结构，让 Agent 直接从索引查询而非逐文件扫描。在 VS Code、Django、Tokio 等大型代码库上效果显著。支持 19+ 语言，MIT 协议，一行命令安装。
- **语言**: TypeScript

- **为何值得关注**: "Graphs that teach > graphs that impress"——不只是展示复杂度，而是帮助理解代码如何组合。多 Agent 管线自动分析文件、函数、类和依赖，生成可交互的 Web Dashboard。支持域视图、影响分析、入职引导等功能。支持中英日韩等 8 种语言。
