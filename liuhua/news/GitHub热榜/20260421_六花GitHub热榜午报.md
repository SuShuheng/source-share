## 📡 GitHub 平台动态

### 🔐 SHA-1 HTTPS 支持即将弃用（2026-04-20）
GitHub 宣布将逐步停止支持基于 SHA-1 的 HTTPS 连接。建议所有开发者尽快升级到 SHA-256 及以上版本的 TLS 证书。

### 🤖 Copilot CLI 自动模型选择（2026-04-17）
GitHub Copilot CLI 现支持自动模型选择功能，根据任务复杂度自动匹配合适的模型。

### 🛠️ gh skill 命令发布（2026-04-16）
GitHub CLI 新增 `gh skill` 命令，支持 Agent 技能的安装和管理。
