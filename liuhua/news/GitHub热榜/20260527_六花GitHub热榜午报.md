- ⭐ 4,857 | 📈 今日 +547
- **核心用途**：一个 Skill 文件，教 LLM 识别和消除 AI 写作的典型痕迹——可预测的短语、结构模式和节奏。
- **为何值得关注**：AI 生成内容有明显的"腔调"，这个项目提供了详细的禁用短语列表、结构模式避免指南和改写前后对比。评分系统从 5 个维度打分（直接性、节奏、信任感、真实感、密度），低于 35/50 分就需修改。对于需要高质量人类风格写作的场景非常实用。
