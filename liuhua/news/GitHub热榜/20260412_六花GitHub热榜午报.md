- 17 个内置工作流（fix-issue、idea-to-pr、smart-pr-review 等），可自定义
- 可视化 DAG 工作流编辑器 + 实时监控面板
- 多平台接入：Web UI、CLI、Telegram、Slack、Discord、GitHub

- 北京大学 OpenBMB 团队出品
