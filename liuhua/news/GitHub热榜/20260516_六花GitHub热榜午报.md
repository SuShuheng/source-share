### 4. 📡 ruvnet/RuView — 今日热榜

**链接**: https://github.com/ruvnet/RuView

**核心用途**: 将普通 WiFi 信号转化为实时空间智能平台。利用 ESP32-S3 ($9) 捕获 WiFi CSI 信号，实现穿墙人体姿态估计（17 个 COCO 关键点）、呼吸心率监测、存在检测、跌倒检测——完全不需要摄像头。

**为何值得关注**: 基于 CMU DensePose From WiFi 研究的开源实现，Rust 编写，1463 个测试通过。无摄像头意味着天然规避 GDPR/HIPAA 隐私合规问题，成本仅 $0-$8/区域。适用于养老院跌倒检测、智能家居、酒店入住管理等场景。
