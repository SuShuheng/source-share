## 🔥 今日爆火项目

### 1. chopratejas/headroom — LLM 上下文压缩层
- **链接**: https://github.com/chopratejas/headroom
- **语言**: Python / TypeScript
- **今日新增**: ⭐ 3,139
- **总星数**: 11,964
- **核心用途**: AI Agent 的上下文压缩层。在工具输出、日志、RAG 分块、对话历史发送到 LLM 之前进行压缩，实现 **60-95% token 缩减**，同时保持答案质量不变。
- **为何值得关注**:
  - 支持 Library / Proxy / MCP Server / Agent Wrap 四种集成方式
  - 提供 6 种压缩算法（SmartCrusher / CodeCompressor / Kompress-base 等）
  - 可逆压缩（CCR）：原始数据本地存储，LLM 按需检索
  - 跨 Agent 记忆共享（Claude Code、Codex、Cursor 等）
  - 在 GSM8K / TruthfulQA 等基准上准确率几乎无损

### 2. NousResearch/hermes-agent — 自我进化的 AI Agent
- **链接**: https://github.com/NousResearch/hermes-agent
- **核心用途**: Nous Research 推出的自我学习 AI Agent。内置学习循环——从经验中创建技能、使用时自我改进、跨会话搜索历史对话、构建用户画像。
- **为何值得关注**:
  - 支持 200+ 模型（OpenRouter、NVIDIA NIM、z.ai/GLM 等）
  - 全平台消息网关（Telegram、Discord、Slack、WhatsApp、Signal）
  - 内置 cron 调度器，支持自然语言自动化任务
  - 可在 $5 VPS 或 GPU 集群上运行，支持 Serverless（Modal / Daytona）
  - 直接与 OpenClaw 竞争，提供 `hermes claw migrate` 迁移工具
