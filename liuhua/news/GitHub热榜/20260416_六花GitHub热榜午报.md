## 📋 热榜项目一览

### 1. 🧠 forrestchang/andrej-karpathy-skills
- **链接**: <https://github.com/forrestchang/andrej-karpathy-skills>
- **核心用途**: 基于 Andrej Karpathy 对 LLM 编程缺陷的观察，提炼出一份 `CLAUDE.md` 规则文件，用于改善 Claude Code 的行为表现。
- **四大原则**: Think Before Coding（先想后写）、Simplicity First（简洁至上）、Surgical Changes（精准修改）、Goal-Driven Execution（目标驱动）。
- **为何值得关注**: Karpathy 亲自指出了 LLM 编程的核心痛点——错误假设、过度设计、乱改代码。这个项目用四条原则系统性解决了这些问题。可直接作为 Claude Code 插件安装。

### 2. 🐚 thedotmack/claude-mem
- **链接**: <https://github.com/thedotmack/claude-mem>
- **核心用途**: Claude Code 的持久记忆压缩系统。自动捕获会话中的工具使用记录，用 AI（Claude agent-sdk）压缩后注入未来会话，实现跨会话上下文延续。
- **亮点**: v6.5.0 版本，支持 Progressive Disclosure（渐进式上下文披露）、Chroma 向量数据库混合搜索、Web UI 内存流查看器、OpenClaw Gateway 集成。
- **为何值得关注**: 解决了 AI 编程助手最大的痛点——每次会话都"失忆"。在 Awesome Claude Code 榜单上有名，社区活跃度极高。

### 3. 🤖 NousResearch/hermes-agent
- **链接**: <https://github.com/NousResearch/hermes-agent>
- **核心用途**: 自我改进型 AI Agent，具备内置学习循环——从经验中创建技能、使用中优化技能、自动持久化知识、搜索历史对话、跨会话建立用户模型。
- **亮点**: 支持多平台（Telegram/Discord/Slack/WhatsApp/Signal/CLI）、多模型切换（200+ 模型）、子 Agent 并行、cron 定时任务、5美元 VPS 即可运行。兼容 agentskills.io 开放标准。
- **为何值得关注**: Nous Research 出品，定位为"唯一具备学习闭环的 Agent"。从 CLI 到全平台消息网关，覆盖面极广。`hermes claw migrate` 命令甚至提供了从 OpenClaw 迁移的路径。

### 4. 🦸 obra/superpowers
- **链接**: <https://github.com/obra/superpowers>
- **核心用途**: 完整的 Agent 驱动软件开发工作流框架，包含可组合的"技能"集合——头脑风暴、Git Worktree 管理、计划编写、子 Agent 驱动开发、TDD、代码审查。
- **亮点**: 支持 Claude Code、Cursor、Codex、OpenCode、Copilot、Gemini 等主流平台。核心理念是"不要急着写代码，先搞清楚要做什么"。
- **为何值得关注**: 代表了 AI 辅助开发方法论的前沿——从编码工具升级为完整的软件工程流程自动化。

### 5. 📊 shiyu-coder/Kronos
- **链接**: <https://github.com/shiyu-coder/Kronos>
- **核心用途**: 首个专为金融 K 线（蜡烛图）设计的开源基础模型。使用专用 Tokenizer 将 OHLCV 数据量化为分层离散 Token，再用自回归 Transformer 进行预训练。
- **亮点**: 被 **AAAI 2026** 接收；覆盖 45+ 全球交易所数据；提供从 mini (4.1M) 到 large (499.2M) 的完整模型家族；支持 BTC/USDT 实时预测 Demo。
- **为何值得关注**: 金融 AI 的细分工突破。不同于通用时序模型，Kronos 专门处理金融数据的高噪声特性。学术论文+开源模型+实战 Demo 三位一体。

### 6. 💰 virattt/ai-hedge-fund
- **链接**: <https://github.com/virattt/ai-hedge-fund>
- **核心用途**: AI 驱动的对冲基金概念验证，用 13+ 位投资大师风格的 Agent 协同分析股票——Buffett、Burry、Ackman、Wood、Munger 等各有策略，配合风险管理 Agent 和组合管理 Agent。
- **亮点**: 多 Agent 辩论式投资决策；每位"投资大师"独立分析后汇总；支持 CLI 和 Web 界面。
- **为何值得关注**: Multi-Agent 在金融决策中的经典实验。虽然仅供教育用途，但展示了 AI Agent 协作框架在复杂决策场景中的潜力。

### 7. 📝 microsoft/markitdown
- **链接**: <https://github.com/microsoft/markitdown>
- **核心用途**: 微软出品的轻量 Python 工具，将各类文件（PDF/PPT/Word/Excel/图片/音频/HTML/ZIP/YouTube/EPub 等）转换为 Markdown 格式。
- **亮点**: 新增 **MCP Server** 支持，可直接与 Claude Desktop 等 LLM 应用集成。对 LLM 友好的 Markdown 输出，Token 效率高。
- **为何值得关注**: 微软官方维护，已成为 LLM 数据处理管道中的标准工具。MCP 集成让它在 AI 工具链中的地位进一步提升。

### 8. 🎙️ jamiepine/voicebox
- **链接**: <https://github.com/jamiepine/voicebox>
- **核心用途**: 开源语音合成工作室。支持声音克隆、语音生成、音效应用，全部本地运行。
- **为何值得关注**: AI 语音工具从云端走向本地的趋势代表，隐私友好且零成本。

### 9. 🏗️ pascalorg/editor
- **链接**: <https://github.com/pascalorg/editor>
- **核心用途**: 基于 React Three Fiber 和 WebGPU 的 3D 建筑编辑器。支持创建和分享 3D 建筑项目。
- **为何值得关注**: WebGPU 技术在专业 3D 应用中的前沿实践。Zustand 状态管理 + IndexedDB 持久化的架构值得学习。
