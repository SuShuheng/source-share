## 🔥 今日爆火项目

### 1. warpdotdev/warp — Agentic 开发环境
- **链接**: https://github.com/warpdotdev/warp
- **今日 Stars**: ~6,175 ⭐
- **语言**: Rust
- **核心用途**: Warp 是一款从终端诞生的 Agentic 开发环境，支持内置 Coding Agent，也可接入 Claude Code、Codex、Gemini CLI 等外部 Agent。OpenAI 作为创始赞助商，Agentic 管理工作流由 GPT 模型驱动。
- **为何值得关注**: 终端 → IDE → Agentic 工作流的演进趋势。开源（AGPL v3 + MIT 双许可），社区贡献活跃，build.warp.dev 可实时观看数千 Oz Agent 分诊 Issue、编写 Spec 和 Review PR。**今日 GitHub Trending 榜首。**

### 2. mattpocock/skills — AI Agent 工程技能库
- **链接**: https://github.com/mattpocock/skills
- **总 Stars**: ~48,512 ⭐ | **今日**: ~3,950 ⭐
- **语言**: Shell
- **核心用途**: 来自 Matt Pocock 的 `.claude` 目录——为"真正的工程师"打造的 AI Agent 技能集。核心技能包括 `/grill-me`（需求对齐）、`/grill-with-docs`（文档驱动的需求挖掘）、`/tdd`（红绿重构 TDD 循环）。支持 Claude Code、Codex 等主流 Agent。
- **为何值得关注**: 直击 AI Coding 的三大痛点——需求不对齐、Agent 过于冗长、代码不工作。强调共享语言（Shared Language）理念来降低 Agent token 消耗。60,000+ 开发者订阅。

### 3. 1jehuang/jcode — 极致性能 Coding Agent Harness
- **链接**: https://github.com/1jehuang/jcode
- **总 Stars**: ~1,802 ⭐ | **今日**: ~670 ⭐
- **语言**: Rust
- **核心用途**: 新一代 Coding Agent Harness，专注于多会话工作流、无限可定制性和极致性能。单会话仅占用 27.8 MB RAM（vs Claude Code 386.6 MB），启动速度 14ms（vs Claude Code 3436.9ms）。内置语义向量记忆系统，每轮对话自动检索相关记忆。
- **为何值得关注**: 性能碾压所有主流 Coding Agent（Claude Code、Cursor、Copilot CLI 等）。10 个并行会话仅 117 MB（vs Claude Code 2300 MB）。用 Rust 编写，为多 Agent 工作流场景量身打造。

### 4. obra/superpowers — Agentic 软件开发方法论
- **链接**: https://github.com/obra/superpowers
- **核心用途**: 一套完整的 Agentic 软件开发方法论和技能框架。从需求挖掘→设计→实现计划→子 Agent 驱动开发→TDD→代码审查→分支收尾，全流程自动化。支持 Claude Code、Codex、Cursor、Copilot、Gemini 等主流平台。
- **为何值得关注**: 不是简单的 Prompt 模板，而是一套完整的 SDLC 自动化流程。子 Agent 驱动开发模式可让 Claude 自主工作数小时不偏离计划。与 mattpocock/skills 形成互补——后者偏技能组件，前者偏完整方法论。

### 5. iamgio/quarkdown — 超级 Markdown 排版系统
- **链接**: https://github.com/iamgio/quarkdown
- **总 Stars**: ~12,893 ⭐ | **今日**: ~799 ⭐
- **语言**: Kotlin
- **核心用途**: 现代化的 Markdown 排版系统，单个项目可编译为论文、书籍、幻灯片、网站或知识库。在 CommonMark/GFM 基础上扩展了图灵完备的函数系统，支持变量、循环、条件语句。VS Code 扩展支持实时预览。
- **为何值得关注**: 让 Markdown 从"纯文本标记"升级为"可编程排版语言"。一次写作，多种输出格式（HTML/PDF/Slides/Docs）。学术写作场景极具潜力。
