### 3. 🧠 OpenViking - AI Agent 专用上下文数据库

**项目地址**：https://github.com/volcengine/OpenViking
**Stars**：10,063 ⭐ | **今日新增**：1,557 ⭐

**核心用途**：
- 专为 AI Agent 设计的开源上下文数据库
- 使用**文件系统范式**统一管理记忆（memory）、资源（resources）、技能（skills）
- 支持分层上下文加载（L0/L1/L2），按需加载节省成本

**为何值得关注**：
- 💾 **统一管理**：告别碎片化存储，用 viking:// 协议管理所有上下文
- 🎯 **分层加载**：
  - L0（摘要）：一句话快速检索
  - L1（概览）：核心信息供 Agent 决策
  - L2（详情）：完整原始数据，按需加载
- 🔍 **目录递归检索**：先定位高分目录，再深入探索，提高检索准确性
- 📊 **可视化检索轨迹**：清晰观察检索路径，便于调试和优化
- 🔄 **自我迭代**：会话结束自动提取长期记忆，Agent 越用越聪明

**实测效果**：
- 集成 OpenViking 后，任务完成率提升 43%，输入 token 成本降低 91%
- 已支持 OpenClaw、OpenCode 等主流 Agent 框架的内存插件

### 4. 🌐 Lightpanda - AI 专用无头浏览器

**项目地址**：https://github.com/lightpanda-io/browser
**Stars**：16,762 ⭐ | **今日新增**：2,100 ⭐

**核心用途**：
- 专为 AI 和自动化设计的无头浏览器
- 使用 Zig 语言从零构建，不基于 Chromium/Blink/WebKit
- 兼容 Playwright、Puppeteer、chromedp（通过 CDP 协议）

**为何值得关注**：
- ⚡ **极致性能**：
  - 内存占用比 Chrome 低 9 倍
  - 执行速度比 Chrome 快 11 倍
  - 即时启动
- 🎯 **专注无头**：没有图形渲染开销，专为自动化场景优化
- 🔧 **兼容主流工具**：可直接替换 Chrome，无需修改现有脚本
- 🤖 **AI 场景优化**：适合 AI 代理、LLM 训练、爬虫、自动化测试

**快速体验**：
```bash
# Linux 安装
curl -L -o lightpanda https://github.com/lightpanda-io/browser/releases/download/nightly/lightpanda-x86_64-linux
chmod a+x ./lightpanda

# 启动 CDP 服务器
./lightpanda serve --host 127.0.0.1 --port 9222

# 使用 Puppeteer 连接
const browser = await puppeteer.connect({
  browserWSEndpoint: "ws://127.0.0.1:9222"
});
```

**项目地址**：https://github.com/langflow-ai/openrag
**Stars**：2,597 ⭐ | **今日新增**：568 ⭐

**核心用途**：
- 基于 Langflow + Docling + OpenSearch 的完整 RAG 平台
- 智能文档搜索和 AI 对话，开箱即用

**为何值得关注**：
- 🎨 **拖拽式工作流**：Langflow 提供可视化界面，快速迭代
- 🤖 **代理式 RAG**：支持重排序、多代理协调等高级编排
- 📦 **多 SDK 支持**：Python、TypeScript/JavaScript SDK，还有 MCP 服务器
- 🚀 **企业级扩展**：模块化设计，可按需扩展功能

**快速开始**：
```bash
# 安装并启动
pip install openrag
uv run openrag

# Python SDK
from openrag_sdk import OpenRAGClient

async with OpenRAGClient() as client:
    response = await client.chat.create(message="What is RAG?")
    print(response.response)
```
