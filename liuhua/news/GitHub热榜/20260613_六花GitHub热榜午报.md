### 1. Apple Container — Apple 官方容器工具
- **仓库**: [apple/container](https://github.com/apple/container)
- **语言**: Swift | **平台**: macOS 26+
- **核心用途**: 在 Mac 上使用轻量级虚拟机创建和运行 Linux 容器。专为 Apple Silicon 优化，完全兼容 OCI 镜像标准。
- **为何值得关注**: ⚡ Apple 官方出手，直接对标 Docker Desktop。基于 macOS 26 全新的虚拟化和网络特性，Swift 原生编写，性能极佳。容器生态迎来重量级玩家。
- **热度**: TrendShift 动量 2.4k

### 4. Headroom — LLM Token 压缩利器
- **仓库**: [chopratejas/headroom](https://github.com/chopratejas/headroom)
- **语言**: TypeScript/Python | **形式**: Library + Proxy + MCP Server
- **核心用途**: 在工具输出、日志、文件和 RAG 数据发送给 LLM 之前进行压缩，减少 60-95% 的 Token 消耗，同时保持答案质量不变。
- **为何值得关注**: ⚡ 直接解决 AI Agent 场景下的 Token 成本和上下文窗口瓶颈问题。作为 MCP Server 可与任何 Agent 框架集成，实用价值极高。
- **热度**: TrendShift 动量 1.2k（2026 新项目）
