## 🔥 今日爆款项目

### 1. affaan-m/ECC ⭐ 182K+
- **链接**: https://github.com/affaan-m/ECC
- **核心用途**: 跨 Agent 框架的性能优化系统，为 Claude Code、Codex、Cursor、OpenCode 等 AI 编程工具提供技能、本能记忆、安全扫描和研究优先开发流程。
- **为何值得关注**: 182K+ Star 的超现象级项目。这不是简单的配置文件，而是完整的 Agent 工作系统——包括 token 优化、跨会话记忆持久化、持续学习和安全扫描。ECC v2.0 还新增了 Hermes 操作器。支持 12+ 语言生态和 7+ 主流 AI 编程工具。
- **AI 相关**: ✅ 核心 AI Agent 工具链

### 2. harry0703/MoneyPrinterTurbo
- **链接**: https://github.com/harry0703/MoneyPrinterTurbo
- **核心用途**: 利用 AI 大模型一键生成高清短视频。只需提供主题或关键词，即可全自动生成视频文案、素材、字幕和背景音乐，合成完整短视频。
- **为何值得关注**: AI 视频生成赛道的标杆项目。支持 OpenAI、DeepSeek、Moonshot、Gemini、Ollama 等十余种模型接入，提供 Web UI 和 API 双模式，Docker 一键部署。
- **AI 相关**: ✅ AI 视频生成

### 3. anthropics/skills
- **链接**: https://github.com/anthropics/skills
- **核心用途**: Anthropic 官方发布的 Agent Skills 标准与示例技能仓库，涵盖创意设计、技术开发、企业沟通和文档处理（docx/pdf/pptx/xlsx）等场景。
- **为何值得关注**: Anthropic 官方定义了 Agent Skills 标准（agentskills.io），这是该标准的参考实现。内含 Claude 文档能力的底层技能源码，也是构建自定义技能的权威模板。
- **AI 相关**: ✅ Agent 技能标准

### 4. OpenMOSS/MOSS-TTS
- **链接**: https://github.com/OpenMOSS/MOSS-TTS
- **核心用途**: 开源语音与声音生成模型家族，覆盖稳定长文本语音、多说话人对话、声音设计、环境音效和实时流式 TTS。
- **为何值得关注**: 5月26日刚发布 MOSS-TTS-v1.5 和 MOSS-SoundEffect-v2.0。新版本支持更强的多语言合成、语音克隆、标点跟随韵律和显式停顿控制。还有 ~100M 参数的 Nano 版本可在 4 核 CPU 上运行。
- **AI 相关**: ✅ 语音合成 / 音频生成

### 5. obra/superpowers
- **链接**: https://github.com/obra/superpowers
- **核心用途**: 完整的 AI 编程 Agent 软件开发方法论与技能框架。从需求讨论、设计评审、TDD 实现到子 Agent 驱动开发，全流程自动化。

- **AI 相关**: ✅ AI 编程方法论

## 🌟 高热度项目

### 6. Lum1104/Understand-Anything
- **链接**: https://github.com/Lum1104/Understand-Anything
- **核心用途**: 将任何代码库、知识库或文档转化为可交互的知识图谱，支持可视化探索、语义搜索和引导式学习。
- **为何值得关注**: 多 Agent 管线自动扫描项目，提取文件/函数/类/依赖关系并构建知识图谱。提供可交互 Dashboard，支持按架构层自动分组、变更影响分析、领域视图等。新加入团队时理解代码库的利器。
- **AI 相关**: ✅ 代码理解 Agent

- **链接**: https://github.com/revfactory/harness
- **核心用途**: Claude Code 的团队架构工厂，一句话即可根据项目自动生成 Agent 团队定义和技能。
- **为何值得关注**: 位于 Claude Code 生态的 L3 元工厂层。提供 6 种团队架构模式（Pipeline、Fan-out/Fan-in、Expert Pool、Producer-Reviewer、Supervisor、Hierarchical Delegation），自动生成 Agent 定义和技能。
- **AI 相关**: ✅ Agent 团队编排

### 8. EveryInc/compound-engineering-plugin
- **链接**: https://github.com/EveryInc/compound-engineering-plugin
- **核心用途**: 复合工程方法论插件——让每一次工程工作都让下一次更轻松。80% 规划+审查，20% 执行。
- **为何值得关注**: 核心理念是"每次工程工作应让后续工作更容易"。提供完整工作流：/ce-brainstorm → /ce-plan → /ce-work → /ce-code-review → /ce-compound，形成学习闭环。
- **AI 相关**: ✅ AI 辅助工程方法论

### 9. hardikpandya/stop-slop
- **链接**: https://github.com/hardikpandya/stop-slop
- **核心用途**: 教 AI 检测和去除写作中的"AI 味"——可预测的短语、结构和节奏模式。
- **为何值得关注**: AI 生成内容的"去 AI 化"需求日益增长。提供被禁短语列表、结构化陈词滥调检测、5 维评分系统（直接性/节奏/信任/真实性/密度），低于 35/50 分自动要求重写。
- **AI 相关**: ✅ AI 写作优化

### 10. Leonxlnx/taste-skill
- **链接**: https://github.com/Leonxlnx/taste-skill
- **核心用途**: 反 Slop 前端框架，提升 AI 生成 UI 的布局、排版、动效和间距质量，告别模板化外观。
- **为何值得关注**: 配合 Codex、Cursor、Claude Code 使用，让 AI 生成的界面不再千篇一律。还包含图像生成技能，可生成参考板（网页/移动端/品牌）。
- **AI 相关**: ✅ AI 前端设计

### 11. microsoft/markitdown
- **链接**: https://github.com/microsoft/markitdown
- **核心用途**: 微软出品的轻量级 Python 工具，将各种文件（PDF、PPT、Word、Excel、图片、音频、HTML 等）转换为 Markdown，专为 LLM 消费优化。
- **为何值得关注**: LLM 的天然接口就是 Markdown。支持 10+ 文件格式，保留文档结构（标题、列表、表格、链接），是 RAG 管线和文本分析的基础工具。
- **AI 相关**: ✅ LLM 数据预处理
