## 🔥 今日爆款项目

### 1. bytedance/UI-TARS-desktop
- **链接**: https://github.com/bytedance/UI-TARS-desktop
- **核心用途**: 开源多模态 AI Agent 技术栈，包含 Agent TARS（CLI/Web UI 多模态 Agent）和 UI-TARS Desktop（桌面端 GUI Agent）
- **为何值得关注**: 字节跳动出品，集成了 GUI Agent + Vision + MCP 工具链，支持浏览器/桌面/终端全场景操作。Agent TARS CLI 已发布 v0.3.0，支持流式工具调用、沙箱隔离执行环境。当前 Trending 榜首。
- **标签**: `#AI-Agent` `#GUI-Agent` `#多模态`

### 2. CloakHQ/CloakBrowser
- **链接**: https://github.com/CloakHQ/CloakBrowser
- **核心用途**: 隐身 Chromium 浏览器，在 C++ 源码层修改了 26 处指纹特征，可作为 Playwright/Puppeteer 的直接替代品
- **为何值得关注**: 能通过所有主流 bot 检测测试，支持持久化 Profile，最新基于 Chromium 146。对自动化爬虫、RPA、反检测场景价值极高。
- **标签**: `#浏览器自动化` `#反检测` `#安全`

### 3. NousResearch/hermes-agent
- **链接**: https://github.com/NousResearch/hermes-agent
- **核心用途**: 自我进化的 AI 编程 Agent，具备持久记忆和自动生成技能的能力
- **为何值得关注**: Nous Research 出品（以开源 LLM 闻名），8100+ Commits 的成熟项目。支持多 AI Provider（OpenRouter/Anthropic/Ollama/vLLM），具备 TUI 和 Web 界面，内置 ACP（Agent Communication Protocol）适配器。核心理念是"与你共同成长的 Agent"。
- **标签**: `#AI-Agent` `#编程助手` `#自我进化`

### 4. decolua/9router
- **链接**: https://github.com/decolua/9router
- **核心用途**: 免费 AI 编程路由器，连接 Claude Code/Cursor/Copilot/Codex/Cline 等编码工具到 40+ AI Provider、100+ 模型，自动降级到免费/低价模型
- **为何值得关注**: 解决 AI 编程工具的 API 成本痛点，支持 RTK（Request Token Keeper）节省 20-40% Token，自动 fallback。554 Commits，Next.js 构建，有完整的多语言支持（含中文）。
- **标签**: `#AI工具` `#成本优化` `#API路由`
