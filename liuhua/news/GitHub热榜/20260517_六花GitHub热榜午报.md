### 3. pranshuparmar/witr ⭐ 16,763 (+411/日)
- **链接**：https://github.com/pranshuparmar/witr
- **语言**：Go
- **核心用途**：回答"为什么这个进程在运行？"的系统诊断工具。不同于 ps/top 只展示状态，witr 追踪进程的完整因果链——从 supervisor、容器、service 到 shell 的启动路径，并提供交互式 TUI 仪表盘。单二进制分发，支持 Linux/macOS/FreeBSD/Windows。
- **为何值得关注**：16.7k stars 说明开发者对"进程为什么在跑"这个痛点的共鸣。填补了传统工具（ps、lsof、systemctl）在因果推理上的空白。DevOps/SRE 日常排障利器。已进入 Homebrew、Conda 等主流包管理器。
