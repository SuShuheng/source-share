## 🔥 今日爆火项目 TOP 10

### 1. 🧠 tinyhumansai/openhuman
- **链接**: https://github.com/tinyhumansai/openhuman
- **Star 趋势**: 📈 Trending #1（今日最火）
- **核心用途**: 开源本地 AI 桌面智能助手，定位"个人 AI 超级智能"
- **亮点**:
  - 118+ 第三方服务一键 OAuth 集成（Gmail、Notion、GitHub、Slack、Calendar 等）
  - Memory Tree + Obsidian Wiki 本地知识库，灵感来自 Karpathy 的 LLM Wiki 模式
  - 20 分钟自动同步数据到本地 SQLite，无需训练等待
  - 内置 TokenJuice 智能压缩层，降低 80% token 消耗
  - 桌面吉祥物 UI，可加入 Google Meet 作为参与者
  - 内置语音（STT + ElevenLabs TTS + 唇形同步）
  - 模型路由自动选推理/快速/视觉模型
  - GNU 开源，Tauri 桌面应用
- **为何值得关注**: 这是目前最接近"个人 AI 操作系统"概念的开源项目。与 Claude Cowork、OpenClaw 对标，主打 UI 优先 + 本地隐私 + 零配置上手。概念很新，但已登上 Trending 榜首。

### 2. 💾 rohitg00/agentmemory
- **链接**: https://github.com/rohitg00/agentmemory
- **核心用途**: AI 编码 Agent 的持久化记忆层
- **亮点**:
  - 支持 Claude Code、Cursor、Gemini CLI、Codex CLI、OpenClaw 等所有主流 Agent
  - 基于 iii 引擎，95.2% 召回率（R@5），减少 92% token 使用
  - 源自 1200+ star 的 GitHub Gist 设计文档
  - npm 包 `@agentmemory/agentmemory` 可直接安装
  - 扩展 Karpathy LLM Wiki 模式：增加置信度评分、生命周期管理、知识图谱、混合搜索
- **为何值得关注**: Agent 记忆是当前 AI Agent 生态最痛的短板之一。这个项目试图标准化 Agent 的长期记忆方案，已获得大量关注。

### 3. 🌐 CloakHQ/CloakBrowser
- **链接**: https://github.com/CloakHQ/CloakBrowser
- **核心用途**: 隐身 Chromium 浏览器，从 C++ 源码级修改指纹，通过所有反爬检测
- **亮点**:
  - 不是配置补丁或 JS 注入，而是真正的 Chromium 二进制级修改
  - 通过 Cloudflare Turnstile 等主流反爬测试
  - Playwright/Puppeteer 的直接替代品，3 行代码即可切换
  - 支持 Python 和 JavaScript，PyPI/npm/Docker 多渠道分发
- **为何值得关注**: 反反爬领域的重磅开源方案，对于自动化测试和爬虫开发者有极高实用价值。
