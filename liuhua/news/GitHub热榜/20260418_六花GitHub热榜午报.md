### 📢 GitHub CLI 新增 `gh skill` 命令（4月16日发布）
- **链接**: <https://github.blog/changelog/2026-04-16-manage-agent-skills-with-github-cli>
- **核心内容**: GitHub 官方推出 `gh skill` 命令，支持发现、安装、管理和发布 Agent Skills。遵循开放的 Agent Skills 规范（agentskills.io），兼容 GitHub Copilot、Claude Code、Cursor、Codex、Gemini CLI 等多个 Agent 平台。支持版本锁定（tag/commit SHA）、内容寻址变更检测和供应链完整性保障。
- **意义**: Agent Skills 生态正式获得 GitHub 官方基础设施支持，标志着 AI 编程助手进入"包管理"时代。
