## 📡 GitHub 官方动态

### Copilot Usage Metrics API 新增 AI 采用阶段分类
- **时间**: 2026-05-29
- **内容**: GitHub Copilot 使用指标 API 新增 `ai_adoption_phase` 字段，将用户分为四个阶段：
  - Phase 0: 无分组
  - Phase 1: Code First（代码补全/IDE Agent）
  - Phase 2: Agent First（单 Agent 表面）

- **意义**: 企业可追踪开发者从"辅助编码"到"多 Agent 协作"的成熟度演进。
