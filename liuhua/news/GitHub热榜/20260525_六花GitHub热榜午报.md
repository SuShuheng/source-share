### 3. Alishahryar1/free-claude-code ⭐ 27.5k (+25.3k/月)

### 5. rohitg00/agentmemory ⭐ 15.8k (+13.7k/月)

## 📡 GitHub 官方动态

### npm Staged Publishing 正式 GA
- **日期**: 2026-05-22
- **要点**: npm CLI 11.15.0 起，支持分阶段发布——包上传到 staging queue，维护者需显式批准后才对外可见。新增 `--allow-file`、`--allow-remote`、`--allow-directory` 安装源控制标志。
- **影响**: 供应链安全重大升级，CI/CD 发布不再直接暴露到 registry。

### GitHub Issues 性能优化
- **日期**: 2026-05-14
- **要点**: 使用 IndexedDB 客户端缓存 + Service Worker + 智能预取策略，将 Issues 导航 HPC（最高优先内容）从 >1s 优化到 <200ms 的"instant"级别。
- **影响**: 对大型项目 Issue 管理体验提升显著，方案可复用到其他数据密集型 Web 应用。
