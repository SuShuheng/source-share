### 3. MLX-VLM — Mac 端视觉语言模型推理与微调工具包

| 项目 | 链接 | 语言 |
|------|------|------|
| Blaizzy/mlx-vlm | [GitHub](https://github.com/Blaizzy/mlx-vlm) | Python |

**核心用途：** 基于 Apple MLX 框架的视觉语言模型（VLM）推理和微调工具包，支持图像、音频、视频多模态输入。

**为何值得关注：**
- 支持 Gemma 4、DeepSeek-OCR、Phi-4 等主流 VLM 模型
- **TurboQuant KV Cache**：3.5-bit 压缩，KV 内存减少 **76%**，支持 512K+ 超长上下文
- **Vision Feature Cache**：多轮对话视觉特征缓存，prompt 处理速度提升 **11x+**
- 内置 OpenAI 兼容的 FastAPI Server（`/v1/chat/completions`）
- 支持 LoRA/QLoRA 微调

## 🛠️ GitHub Engineering 精选

### Diff Lines 性能优化：从 8 组件到 2 组件
- **来源：** [GitHub Engineering Blog](https://github.blog/engineering/architecture-optimization/the-uphill-climb-of-making-diff-lines-performant/)
- **亮点：** PR "Files Changed" 标签页全面重构，每行 diff 的 React 组件从 8 个精简到 2 个
- **效果：** 显著降低 DOM 节点数、JS 堆内存占用和 INP 延迟，尤其改善大型 PR 的审查体验
- **技术路线：** 简化组件树 + 虚拟化降级 + 基础渲染优化
