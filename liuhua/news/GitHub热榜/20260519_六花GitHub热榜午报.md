## 🔥 今日爆款项目

### 1. tinyhumansai/openhuman ⭐ 日增 1,000+

- **链接**: https://github.com/tinyhumansai/openhuman
- **核心用途**: 开源个人 AI 超级智能体，定位为"你的个人 AI 超级大脑"。提供桌面客户端，内置 118+ 第三方集成（Gmail、Notion、GitHub、Slack 等），通过一键 OAuth 连接，20 分钟自动拉取数据到本地 Memory Tree。
- **关键特性**:
  - Memory Tree + Obsidian Wiki 本地知识库（受 Karpathy 启发）
  - TokenJuice 智能压缩，降低 80% token 成本和延迟
  - 桌面吉祥物角色，可加入 Google Meet 作为参与者
  - 内置模型路由（reasoning / fast / vision 自动切换）
  - 原生语音支持（STT + ElevenLabs TTS）
  - 支持 Ollama 本地模型
- **为何值得关注**: 首个"分钟级冷启动"的 Agent 框架，跳过了传统 Agent 需要数周学习用户上下文的瓶颈。Product Hunt 当日 Top 产品，GitHub Trending 排名第一。
- **语言**: TypeScript / Rust (Tauri) | **协议**: GNU

### 2. HKUDS/CLI-Anything

- **链接**: https://github.com/HKUDS/CLI-Anything
- **核心用途**: "让所有软件原生支持 Agent"。通过自动生成 CLI 包装器，将任何 GUI 软件转化为 AI Agent 可直接调用的命令行工具。配有 CLI-Hub 社区（clianything.cc），用户可浏览、安装、管理社区构建的 CLI。
- **关键特性**:
  - 一键将任意软件封装为 Agent 可用的 CLI
  - 兼容 OpenClaw、Cursor、Claude Code 等主流 Agent 框架
  - 社区驱动的 CLI-Hub，PR 即发布
  - `pip install cli-anything-hub` 即可使用
- **为何值得关注**: Agent 生态的关键基础设施——解决了 Agent 无法直接操作传统 GUI 软件的痛点。来自港大数据科学实验室（HKUDS），学术背景扎实。
- **语言**: Python

### 3. OpenAI Symphony（开源）

- **链接**: https://github.com/openai/symphony
- **核心用途**: OpenAI 开源的自主编码 Agent 编排框架。通过连接 Linear 等项目管理工具，自动为每个 Issue 派发 Codex Agent，执行编码任务并提交 PR。
- **关键特性**:
  - Linear Board 驱动的自动化工作流
  - 每个 Issue 自动启动独立 Codex Agent 实例
  - 完成后自动创建 Pull Request
  - SPEC.md 规范定义 Agent 编排标准
- **为何值得关注**: OpenAI 首次公开其内部 Agent 编排方案，代表了"AI 自主开发"的前沿实践。对理解大规模 Agent 协作有重要参考价值。
- **语言**: TypeScript / Python

- **链接**: https://github.blog/engineering/architecture-optimization/from-latency-to-instant-modernizing-github-issues-navigation-performance/
