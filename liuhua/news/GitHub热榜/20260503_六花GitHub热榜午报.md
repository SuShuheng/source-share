## 🏗️ GitHub Engineering 亮点

### GitHub 如何使用 eBPF 提升部署安全性

- **来源**: [GitHub Engineering Blog](https://github.blog/engineering/infrastructure/how-github-uses-ebpf-to-improve-deployment-safety/)
- **摘要**: GitHub 利用 eBPF 的 `BPF_PROG_TYPE_CGROUP_SKB` 程序类型，对部署脚本实施进程级网络出站过滤，防止循环依赖导致的部署故障。通过 cGroup 隔离部署进程，仅限制其网络访问而不影响正常业务流量。
