### 2. NousResearch/hermes-agent
**链接**: https://github.com/NousResearch/hermes-agent  
**核心用途**: 具备自我学习循环的 AI Agent——能从经验中自动创建技能、使用时自我改进、跨会话构建用户画像。支持 200+ 模型切换（OpenRouter）、Telegram/Discord/Slack 多平台接入、内置 cron 定时任务和子 Agent 并行执行。  
**为何值得关注**: 这是 Nous Research 的重磅开源 Agent 项目，与 OpenClaw 架构理念高度相似（子 Agent、cron、多平台网关）。亮点在于"学习循环"设计——Agent 会主动创建和改进技能，跨会话记忆搜索，配合 Honcho 用户建模系统。可在 $5 VPS 上运行。

### 3. apple/container ⭐ 日增 ~1.9k
**链接**: https://github.com/apple/container  
**核心用途**: Apple 官方容器工具——在 Mac 上使用轻量级虚拟机创建和运行 Linux 容器，用 Swift 编写，针对 Apple Silicon 优化。支持 OCI 标准镜像，可从任何标准容器注册表拉取和推送。  
**为何值得关注**: Apple 终于正式进军容器生态。基于 macOS 26 新虚拟化特性，提供原生、无 Docker 依赖的容器体验。对 Mac 开发者来说是 Docker Desktop 的轻量替代品，OCI 兼容意味着无缝对接现有工作流。  
**标签**: `#Container` `#Apple` `#Swift`
