## 🔥 今日热榜 Top 7

### 1. 🏆 Yeachan-Heo/oh-my-codex — OpenAI Codex 的工作流增强层

- **链接**: <https://github.com/Yeachan-Heo/oh-my-codex>
- **语言**: TypeScript / Node.js
- **核心用途**: 为 OpenAI Codex CLI 添加 hooks、agent 团队协作、HUD 和持久化工作流。提供 `$deep-interview`（需求澄清）→ `$ralplan`（计划审批）→ `$team`（并行执行）/ `$ralph`（持续完成）的标准流程。
- **为何值得关注**: 不替代 Codex 而是增强它——将 Codex 从"单次对话工具"提升为可复用工作流引擎。支持 tmux 团队模式和 OpenClaw 集成，MIT 协议，多语言文档（含中文）。
- **标签**: #AI #Agent #Codex #工作流

### 2. 🥈 onyx-dot-app/onyx — 开源 AI 平台（RAG + Agent + Deep Research）

- **链接**: <https://github.com/onyx-dot-app/onyx>
- **语言**: Python / TypeScript
- **核心用途**: 面向企业的开源 AI 平台，核心功能包括 Agentic RAG、深度研究（Deep Research 排行榜第一）、自定义 Agent、网页搜索、代码执行、语音模式、图片生成、Artifacts 等。支持 50+ 数据连接器和 MCP 协议。
- **为何值得关注**: 功能最全面的开源 AI 平台之一，Deep Research 基准测试 2026 年 2 月排名第一。Lite 模式只需 1GB 内存即可运行，Docker 一键部署。兼容所有主流 LLM（OpenAI、Anthropic、Gemini、Ollama 等）。
- **标签**: #AI #RAG #Agent #企业级 #开源

### 3. 🥉 google-research/timesfm — Google 时间序列基础模型

- **链接**: <https://github.com/google-research/timesfm>
- **语言**: Python / JAX / PyTorch
- **核心用途**: Google Research 开发的时间序列预测基础模型（TimesFM 2.5）。200M 参数，支持 16K 上下文长度，可通过可选的 30M 分位数头进行连续分位数预测。
- **为何值得关注**: ICML 2024 论文实现，已集成到 Google BigQuery。相比 2.0 版本，参数从 500M 降到 200M 但上下文长度从 2048 提升到 16K。新增 AGENTS.md 支持和 OpenClaw 技能文件。
- **标签**: #AI #时间序列 #Google #预测模型

### 4. dmtrKovalenko/fff.nvim — 极速模糊文件搜索（AI Agent + Neovim）

- **链接**: <https://github.com/dmtrKovalenko/fff.nvim>
- **语言**: Rust / Lua
- **核心用途**: "Freakin Fast Fuzzy File Finder"——专为 AI Agent 和 Neovim 设计的模糊文件搜索工具。内置记忆系统，基于 frecency、git status、文件大小、定义匹配等因素优化排序。
- **为何值得关注**: 可作为 MCP 服务连接 Claude Code、Codex、OpenCode 等 AI 工具。显著减少 Agent 搜索文件的时间和 token 消耗。在 Linux 内核仓库（10万文件、8GB）上实测极速。
- **标签**: #AI #开发工具 #Neovim #MCP #性能

### 5. f/prompts.chat — 全球最大开源 AI Prompt 库

- **链接**: <https://github.com/f/prompts.chat>
- **语言**: Web (Next.js)
- **核心用途**: 原名 "Awesome ChatGPT Prompts"，全球最大开源 prompt 库（143K+ GitHub Stars）。支持自部署、MCP 服务器模式、Claude 插件集成。
- **为何值得关注**: Harvard、Columbia 引用，Forbes 报道，HuggingFace 最受欢迎数据集。新增 kids 模式（8-14 岁儿童 AI 互动学习）和完整自托管方案。GitHub Staff Pick。
- **标签**: #AI #Prompt #开源 #社区

### 6. siddharthvaddem/openscreen — 开源 Screen Studio 替代品

- **链接**: <https://github.com/siddharthvaddem/openscreen>
- **语言**: TypeScript / Electron
- **核心用途**: 免费开源的产品演示录制工具（Screen Studio 替代品）。支持屏幕录制、自动/手动缩放、注释（文字、箭头、图片）、运动模糊、裁剪、速度调节、多比例导出。
- **为何值得关注**: Screen Studio 月费 $29，OpenScreen 完全免费且支持商业用途。跨平台（macOS / Windows / Linux），MIT 协议。虽处于 beta 阶段但核心功能完整。
- **标签**: #开发工具 #录屏 #演示 #开源替代

### 7. sherlock-project/sherlock — 跨 400+ 社交网络的用户名追踪工具

- **链接**: <https://github.com/sherlock-project/sherlock>
- **语言**: Python
- **核心用途**: 通过用户名在 400+ 社交网络中追踪账号。支持 Tor、代理、CSV/XLSX 导出。
- **为何值得关注**: 经典 OSINT 工具持续更新，v0.14.3 支持 Docker 和 Apify 云端运行。适合安全研究人员和数字取证。
- **标签**: #安全 #OSINT #隐私 #社交网络
