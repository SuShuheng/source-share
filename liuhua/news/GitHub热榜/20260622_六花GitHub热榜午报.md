## 🔥 今日爆款项目 TOP 10

### 1. palmier-io/palmier-pro — AI 原生视频编辑器
- **链接**: <https://github.com/palmier-io/palmier-pro>
- **核心用途**: macOS 平台的 AI 视频编辑器，Swift 原生构建，对标 Premiere Pro。内置 Seedance、Kling、Nano Banana Pro 等生成式 AI 模型，支持通过 MCP 协议连接 Claude Code / Codex / Cursor，让 AI Agent 直接操作时间线。
- **为何值得关注**: 将 AI Agent 真正嵌入视频创作工作流的先驱产品。YC S24 孵化，编辑器部分完全开源（GPLv3），生成式 AI 功能需订阅。代表"AI + 创意工具"融合的新范式。

### 2. calesthio/OpenMontage — 开源 Agent 视频制作系统
- **链接**: <https://github.com/calesthio/OpenMontage>
- **核心用途**: 首个开源的 Agentic 视频制作系统。包含 12 条流水线、52 个工具、500+ Agent 技能。用自然语言描述需求，Agent 自动完成调研、脚本、素材生成、剪辑和合成。支持 Veo、Kling v3、FLUX 等多种生成模型。
- **为何值得关注**: 示例中用纯 AI 流程制作的电影级预告片、皮克斯风格动画短片（成本仅 $1.33）、产品广告（成本 $0.69），展示了 Agentic 视频制作的可行性和极低成本。AGPLv3 开源。

### 3. DeusData/codebase-memory-mcp — 代码知识图谱 MCP 服务器
- **链接**: <https://github.com/DeusData/codebase-memory-mcp>
- **核心用途**: 高性能代码智能引擎，将整个代码库索引为持久化知识图谱。Linux 内核（28M 行代码、75K 文件）3 分钟完成索引，查询响应 <1ms。支持 158 种语言（tree-sitter AST），9 种语言的 Hybrid LSP 语义类型解析。14 个 MCP 工具，零依赖单一静态二进制。
- **为何值得关注**: 发表 arXiv 论文（2603.27277），在 31 个真实仓库评测中：83% 回答质量、10× 更少 token、2.1× 更少工具调用。SLSA 3 安全等级，70+ 杀软扫描。AI 编程助手的"记忆中枢"。

### 4. jamiepine/voicebox — 开源 AI 语音工作室
- **链接**: <https://github.com/jamiepine/voicebox>
- **核心用途**: 本地优先的 AI 语音工作室，ElevenLabs + WisprFlow 的开源替代品。支持语音克隆、23 种语言、7 种 TTS 引擎（Qwen3-TTS、Chatterbox 等）、全局听写热键、MCP Agent 语音输出。完全本地运行，隐私零泄露。
- **为何值得关注**: 将语音输入（STT）和语音输出（TTS）统一在一个应用中，并通过本地 LLM 桥接。任何支持 MCP 的 AI Agent（Claude Code、Cursor、Cline）都能一键获得自定义语音。TrendShift 上榜项目。

### 5. tursodatabase/turso — Rust 编写的 SQLite 兼容进程内数据库
- **链接**: <https://github.com/tursodatabase/turso>
- **核心用途**: 用 Rust 从头重写的进程内 SQL 数据库，完全兼容 SQLite 语法和文件格式。支持 MVCC 并发写入（`BEGIN CONCURRENT`）、CDC 变更数据捕获、向量搜索、io_uring 异步 I/O。提供 Go/JS/Python/Java/.NET/Rust/WASM 多语言绑定。
- **为何值得关注**: SQLite 生态最重大的 Rust 重写项目。向量搜索 + 增量计算（DBSP）+ 全文搜索（tantivy）的组合在嵌入式数据库中极为罕见。Linux/macOS/Windows/浏览器全平台覆盖。

### 6. withastro/flue — Astro 团队出品的 Agent 框架
- **链接**: <https://github.com/withastro/flue>
- **核心用途**: 不是又一个 SDK，而是 TypeScript Agent 编排框架。提供会话管理、工具系统、技能复用、沙箱执行、持久化恢复、子 Agent 委派等能力。支持部署到 Node.js、Cloudflare Workers、GitHub Actions 等多平台。
- **为何值得关注**: Astro 团队（百万级下载量框架）进军 AI Agent 领域的产品。不是 API 封装，而是"给任何模型一个完整工作环境"的 Harness 架构思路——和 Claude Code / Codex 同一设计哲学，但开源可编程。

### 7. Kilo-Org/kilocode — 多平台 AI 编程 Agent
- **链接**: <https://github.com/Kilo-Org/kilocode>
- **核心用途**: VS Code / JetBrains / CLI / Cloud 全平台覆盖的 AI 编程 Agent。500+ 模型可选，支持中途切换，按模型提供商价格计费零加价。20+ 种语言界面翻译。
- **为何值得关注**: 开源 AI 编程助手中覆盖面最广的产品之一，CLI + IDE + Cloud + 代码审查一站式。零加价模式对开发者极具吸引力。

### 8. twentyhq/twenty — AI 时代的开源 CRM
- **链接**: <https://github.com/twentyhq/twenty>
- **核心用途**: 第一名开源 CRM，定位为"AI 原生的 Salesforce 替代品"。用代码定义对象、字段和视图，支持自定义 Agent 和逻辑函数。提供 CLI 工具 `create-twenty-app` 快速搭建。
- **为何值得关注**: 连续多日上榜，将 CRM 从"配置驱动"升级为"代码驱动"，让技术团队像版本控制代码一样管理业务逻辑。二十（Twenty）已快速成长为开源 CRM 生态标杆。

### 9. chopratejas/headroom — LLM Token 压缩工具
- **链接**: <https://github.com/chopratejas/headroom>
- **核心用途**: 在工具输出、日志、文件和 RAG 文本送入 LLM 之前进行压缩。减少 60-95% token 消耗，保持相同答案质量。提供库、代理和 MCP 服务器三种形态。
- **为何值得关注**: Token 成本是 AI 应用的核心痛点。这个项目从源头解决"上下文太长"问题，作为 MCP 服务器可无缝集成到现有 Agent 工具链。

### 10. mattpocock/skills — 工程师实战 Agent 技能集
- **链接**: <https://github.com/mattpocock/skills>
- **核心用途**: Matt Pocock（Total TypeScript 创始人）的个人 Agent 技能库，包含 `/grill-me`（需求对齐）、`/grill-with-docs`（代码+需求对齐）、`/triage`（工单分类）等实战技能。强调小而精、可组合、易定制。
- **为何值得关注**: 不是 GSD/BMAD/Spec-Kit 等重量级流程框架，而是"反框架"的极简技能集。6 万订阅者的 Newsletter 背书，代表了 Agent 技能设计从"大而全"到"小而美"的趋势。
