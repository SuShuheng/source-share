## 🔥 今日热门项目

### 1. mvanhorn/last30days-skill
- **链接**: https://github.com/mvanhorn/last30days-skill

- **核心用途**: AI Agent 搜索引擎技能，跨 Reddit、X、YouTube、HN、Polymarket 等十余个平台并行检索，按社交互动数据（点赞、投票、真金白银）打分，合成研究简报。
- **为何值得关注**: 🏆 **GitHub Today #1**。v3 版本新增智能预研（自动解析人名→社交账号→社区）、可分享 HTML 简报、跨源聚类合并、竞品对比单次跑通等功能。代表了 AI Agent 从"工具调用"到"自主研究"的范式升级。
- **标签**: `#AI_Agent` `#搜索` `#社交信号`

- **为何值得关注**: Daniel Miessler（安全领域知名博主）的项目。PAI v5.0.0 是该项目历史上最大版本更新，从"AI 扩展器"进化为"生活操作系统"。体现了 AI Agent 向个人生活管理渗透的趋势。

### 5. santifer/career-ops
- **链接**: https://github.com/santifer/career-ops
- **语言**: JavaScript (Node.js + Go)
- **核心用途**: 基于 Claude Code 的 AI 求职系统。自动扫描 45+ 公司招聘页，评估岗位匹配度（A-F 评分），生成 ATS 优化简历 PDF，批量并行处理。
- **为何值得关注**: 被 WIRED 和 Business Insider 报道。作者用该系统评估了 740+ 职位、生成 100+ 定制简历并成功拿到 Head of Applied AI 岗位。是 AI Agent 在求职场景的最佳实践案例。
- **标签**: `#AI_求职` `#Claude_Code` `#自动化`

### 6. hyperlight-dev/hyperlight
- **链接**: https://github.com/hyperlight-dev/hyperlight
- **语言**: Rust
- **核心用途**: 轻量级虚拟机管理器 (VMM)，用于在微型 VM 中安全执行不受信任的代码。毫秒级启动，微秒级函数调用，无 Guest OS。
- **为何值得关注**: CNCF 沙箱项目。对 AI Agent 生态至关重要——为 Agent 生成的代码提供 hypervisor 级别的安全沙箱。支持 KVM/MSHV/WHP，可嵌入 Rust 应用。是 AI 安全基础设施的关键组件。
- **标签**: `#安全` `#虚拟化` `#CNCF` `#AI安全`
