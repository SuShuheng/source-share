## 🔥 今日热榜项目

### 1. siddharthvaddem/openscreen
- 🔗 <https://github.com/siddharthvaddem/openscreen>
- 📝 **核心用途**：开源免费的 Screen Studio 替代品，用于制作精美的产品演示和操作 walkthrough 视频
- ⚡ **为何关注**：Screen Studio 月费 $29，而 OpenScreen 完全免费（MIT 协议）、无水印、可商用。支持屏幕录制、自动/手动缩放、音频捕获、运动模糊、注释标注、裁剪剪辑等功能。基于 Electron + React + TypeScript 构建，跨平台支持 macOS/Windows/Linux。项目虽仍处于 beta，但核心功能已基本可用。
- 🏷️ 工具 · 录屏 · 开源替代

### 2. Yeachan-Heo/oh-my-codex (OMX)
- 🔗 <https://github.com/Yeachan-Heo/oh-my-codex>
- 📝 **核心用途**：OpenAI Codex CLI 的工作流增强层，提供标准化的 `$deep-interview` → `$ralplan` → `$team`/`$ralph` 工作流
- ⚡ **为何关注**：为 Codex CLI 添加了角色管理、技能复用、多 Agent 协同（tmux team runtime）和持久化状态管理（`.omx/`）。相当于给 Codex 加上了 oh-my-zsh 级别的工作流增强。多语言文档支持（含简体中文），社区活跃。
- 🏷️ AI Agent · Codex · 工作流

### 3. asgeirtj/system_prompts_leaks
- 🔗 <https://github.com/asgeirtj/system_prompts_leaks>
- 📝 **核心用途**：系统性收集主流 AI 产品的 System Prompt，涵盖 ChatGPT (GPT-5.4/5.3)、Claude (Opus/Sonnet 4.6)、Gemini (3.1 Pro/3 Flash)、Grok (4.2/4)、Perplexity 等
- ⚡ **为何关注**：对理解各大模型厂商如何设计系统指令、安全策略和工具调用具有极高参考价值。收录了 Claude Code、Codex CLI、Gemini CLI 等编程助手的完整 prompt。持续更新，社区 PR 活跃。对于 prompt engineering 研究者和安全研究人员来说是宝藏级资源。
- 🏷️ AI · Prompt Engineering · 安全研究

## 🌟 GitHub 官方动态

### 4. GitHub Copilot SDK 公开预览 ⭐重大
- 🔗 <https://github.com/github/copilot-sdk>
- 📝 **核心用途**：将 Copilot 的 Agent 能力嵌入到自有应用中的多语言 SDK（TypeScript/Python/Go/.NET/Java）
- ⚡ **为何关注**：这是 GitHub 本周最重要的发布。开发者无需自建 AI 编排层，即可获得工具调用、流式响应、文件操作、多轮对话等能力。支持自定义 Agent/Tool、细粒度 System Prompt 定制、OpenTelemetry 追踪、BYOK（自带 OpenAI/Azure/Anthropic API Key）。所有 Copilot 订阅层级（含 Free）均可使用。
- 📅 发布：2026-04-02

### 5. GPT-5.1 Codex 系列模型退役
- 🔗 <https://github.blog/changelog/2026-04-03-gpt-5-1-codex-gpt-5-1-codex-max-and-gpt-5-1-codex-mini-deprecated>
- 📝 **内容**：GPT-5.1-Codex / Codex-Max / Codex-Mini 已于 4月1日在所有 Copilot 体验中退役，建议迁移至 GPT-5.3-Codex
- ⚡ **影响**：使用旧模型的 CI/CD 工作流需及时更新
- 📅 发布：2026-04-03

### 6. Copilot Cloud Agent 支持签名 Commit
- 🔗 <https://github.blog/changelog/2026-04-03-copilot-cloud-agent-signs-its-commits>
- 📝 **内容**：Cloud Agent 现在对每次 commit 签名，在 GitHub 上显示为 "Verified"
- ⚡ **影响**：可在启用了"Require signed commits"分支保护规则的仓库中使用 Cloud Agent

### 7. Copilot Cloud Agent 防火墙支持组织级管理
