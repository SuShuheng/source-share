## 🔥 爆火项目

### 1. mvanhorn/last30days-skill
**链接**: https://github.com/mvanhorn/last30days-skill  
**核心用途**: AI Agent 技能，可跨 10+ 平台研究任何话题并生成综合报告

**亮点功能**:
- 支持 Reddit, X, Bluesky, YouTube, TikTok, Instagram, Hacker News, Polymarket 等平台
- **v2.9.5 新增**: Bluesky/AT Protocol 支持、对比模式（X vs Y）、项目级配置
- 自动保存研究报告到 `~/Documents/Last30Days/`
- 支持 Polymarket 预测市场数据，获取真金白银的市场判断

**为何值得关注**: 这是 AI Agent 生态中的"研究技能"，可与 Claude Code、OpenAI Codex、Gemini CLI 集成。特别适合需要快速了解某个话题 30 天内动态的场景，如 Prompt 研究和趋势分析。

### 3. GitHub Copilot for Jira 公开预览增强
**链接**: https://github.blog/changelog/2026-03-25-github-copilot-for-jira-public-preview-enhancements  
**发布时间**: 2026-03-25

**新增功能**:

### 4. 隐私政策更新：Copilot 数据用于模型训练
**链接**: https://github.blog/changelog/2026-03-25-updates-to-our-privacy-statement-and-terms-of-service-how-we-use-your-data  
**生效时间**: 2026-04-24

**核心变更**:
- Copilot Free/Pro/Pro+ 用户的交互数据（输入、输出、代码片段）将用于 AI 模型训练
- 用户可在设置中**主动退出**（opt-out）
- Copilot Business 和 Enterprise 用户不受影响

**为何值得关注**: 数据隐私和 AI 训练的边界问题。使用 Copilot 的开发者需要关注自己的代码是否会被用于训练。
