## 🔥 今日焦点

### 1. zilliztech/claude-context
**链接**: https://github.com/zilliztech/claude-context  
**核心用途**: 为 Claude Code 等 AI 编码助手提供语义代码搜索的 MCP 插件，将整个代码库作为上下文。  
**为何值得关注**: 基于 Milvus 向量数据库实现代码语义检索，支持百万行级代码库的精准上下文注入。兼容 Claude Code、Cursor、Windsurf、VS Code 等 10+ 主流 AI 编码工具，大幅降低大代码库的 token 消耗成本。这是 MCP 生态中解决"代码上下文瓶颈"的重要基础设施。  
**标签**: `#MCP` `#AI编码` `#向量搜索`

### 2. HKUDS/RAG-Anything
**链接**: https://github.com/HKUDS/RAG-Anything  
**核心用途**: 下一代多模态 RAG 框架，支持文本、图像、表格、公式、图表等全模态文档处理。  
**为何值得关注**: 香港大学团队出品，基于 LightRAG 构建，解决了传统 RAG 系统仅能处理文本的局限。支持 VLM 增强查询、上下文感知处理、多模态联合检索。已发表技术报告 (arXiv:2510.12323)，GitHub 突破 1k stars。  
**标签**: `#RAG` `#多模态` `#学术研究`

### 3. KeygraphHQ/shannon
**链接**: https://github.com/KeygraphHQ/shannon  
**核心用途**: 自主白盒 AI 渗透测试工具，分析源代码→识别攻击面→执行真实漏洞利用→生成 PoC 报告。  
**为何值得关注**: 在 OWASP Juice Shop 上发现 20+ 漏洞（含认证绕过和数据库泄露）。支持全自动操作（含 2FA/TOTP/SSO 登录），仅报告可利用的漏洞并提供复现步骤。代表了"AI 安全测试即服务"的新趋势。  
**标签**: `#AI安全` `#渗透测试` `#DevSecOps`

### 4. AIDC-AI/Pixelle-Video
**链接**: https://github.com/AIDC-AI/Pixelle-Video  
**核心用途**: AI 全自动短视频引擎——输入主题，AI 自动创作文案→生成配图→TTS 配音→合成视频。  
**为何值得关注**: 阿里 AIDC 团队出品，支持 Windows 一键整合包（无需 Python 环境），集成 ComfyUI 工作流、多种 LLM（通义千问/GPT/DeepSeek）、声音克隆、BGM 配乐。降低了短视频生产的完整技术门槛。  
**标签**: `#AI视频` `#内容生成` `#ComfyUI`

### 5. vercel-labs/skills
**链接**: https://github.com/vercel-labs/skills  
**核心用途**: 开源 Agent Skills 生态的 CLI 工具，支持向 40+ AI 编码工具分发技能包。  
**为何值得关注**: Vercel 实验室出品，一条命令 (`npx skills add`) 即可向 Claude Code、Codex、Cursor 等工具注入技能。支持 GitHub/GitLab/本地路径，项目级或全局安装，符号链接或复制模式。标志着 AI Agent 技能标准化和共享化的开端。  
**标签**: `#Agent生态` `#Vercel` `#工具链`

## 📰 GitHub 官方动态

### Copilot Web 端增强调试能力
**链接**: https://github.blog/changelog/2026-04-23-better-debugging-with-github-copilot-on-the-web
