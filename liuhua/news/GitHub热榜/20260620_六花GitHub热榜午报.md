## 🏆 热榜项目精选

### 1. 🥇 zai-org/GLM-5 — 智谱开源旗舰大模型 GLM-5 系列

- **链接**: https://github.com/zai-org/GLM-5
- **语言**: Python
- **核心用途**: 智谱 AI 开源的旗舰大模型，包含 GLM-5 / GLM-5.1 / GLM-5.2 三个版本。744B 参数（40B 活跃），MoE 架构 + DeepSeek Sparse Attention，GLM-5.2 支持稳定 1M token 上下文。
- **为何值得关注**:
  - GLM-5.2 在 Terminal-Bench 2.1 达到 81.0 分，逼近 Claude Opus 4.8（85.0），开源模型最强
  - 1M token 稳定上下文 + IndexShare 架构创新，每 4 层共享 indexer，1M 上下文下 FLOPs 降低 2.9×
  - 针对长周期 Agent 任务设计，Vending Bench 2 排名开源 #1
  - 支持 SGLang / vLLM / Transformers / KTransformers 多框架部署

### 2. 🥈 google-research/timesfm — Google 时序基础模型

- **链接**: https://github.com/google-research/timesfm
- **语言**: Python
- **核心用途**: Google Research 开发的时间序列预测基础模型，最新 TimesFM 2.5 仅 200M 参数，支持 16k 上下文长度和连续分位数预测。
- **为何值得关注**:
  - 参数量从 500M 降至 200M，性能不降反升
  - 已集成进 BigQuery ML、Google Sheets、Vertex Model Garden
  - 新增 LoRA 微调示例（HuggingFace Transformers + PEFT）
  - `pip install timesfm[torch]` 即装即用

---

### 3. 🥉 DeusData/codebase-memory-mcp — AI 编程 Agent 代码智能引擎

- **链接**: https://github.com/DeusData/codebase-memory-mcp
- **语言**: C（零依赖纯 C 实现）
- **核心用途**: 面向 AI 编程 Agent 的代码智能 MCP 服务，基于 tree-sitter 全量索引代码库，生成函数/类/调用链/HTTP 路由的知识图谱。支持 158 种语言，单二进制即装即用。
- **为何值得关注**:
  - Linux 内核（28M 行，75K 文件）3 分钟完成索引
  - 结构化查询响应 < 1ms，比逐文件搜索节省 120× token
  - 自动检测并配置 11 种编程 Agent（Claude Code、Codex、Gemini CLI、OpenClaw 等）
  - 内置 3D 知识图谱可视化 UI，有论文支撑（arXiv:2603.27277）

### 4. alibaba/zvec — 阿里巴巴开源轻量级向量数据库

- **链接**: https://github.com/alibaba/zvec
- **语言**: C/C++
- **核心用途**: 开源进程内嵌向量数据库，轻量极速，已在阿里集团内部生产验证。v0.5.0（6月12日发布）新增全文搜索、混合检索和 DiskANN 磁盘索引。
- **为何值得关注**:
  - 十亿级向量毫秒级搜索
  - 新增原生全文搜索 + 混合检索（向量 + 全文 + 标量过滤）
  - DiskANN 索引大幅降低内存占用
  - 多语言 SDK：Python / Node.js / Go / Rust / Dart，`pip install zvec` 即用

### 6. Lightricks/LTX-2 — 音视频同步生成基础模型

- **链接**: https://github.com/Lightricks/LTX-2
- **语言**: Python
- **核心用途**: 首个基于 DiT 的音视频基础模型，在单一模型中集成音视频同步生成、高保真输出、多种性能模式。支持文生视频、图生视频、音频驱动视频等。
- **为何值得关注**:

  - 支持 FP8 量化 + FlashAttention 4 优化
  - 开源权重 + 完整代码，支持本地部署

### 10. n0-computer/iroh — Rust 模块化网络栈

- **链接**: https://github.com/n0-computer/iroh
- **语言**: Rust
- **核心用途**: 基于公钥拨号的模块化网络栈，用 QUIC 替代传统 IP 寻址。支持打洞直连 + 中继回退，内含 blob 传输、发布订阅、KV 存储等可组合协议。
- **为何值得关注**:

  - 内置打洞 + QUIC 加密，性能持续监测
  - iroh-blobs 支持从 KB 到 TB 级内容寻址传输
  - 活跃的 Rust 生态，FFI 绑定支持多语言
