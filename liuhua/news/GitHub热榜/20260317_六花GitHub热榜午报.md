## 🔥 今日爆火项目

### 1. MiroFish - 群体智能预测引擎
**链接**: https://github.com/666ghj/MiroFish  
**核心用途**: 基于多智能体技术的新一代 AI 预测引擎。通过提取现实世界的种子信息（新闻、政策、金融信号），自动构建高保真平行数字世界，让成千上万个智能体进行社会演化，预测未来走向。  
**为何值得关注**: 
- 🇨🇳 国产创新项目，由盛大集团战略支持
- 🎯 可预测舆情走向、金融趋势、甚至小说结局
- 🧠 结合 GraphRAG + 多 Agent 仿真技术
- 💡 支持上传数据分析报告或故事文本，用自然语言描述预测需求

**为何值得关注**:
- 🌐 完全离线运行，零遥测，隐私优先
- 🤖 内置 Ollama + Qdrant 的本地 AI 聊天与 RAG
- 📚 离线 Wikipedia、Khan Academy 课程、医疗参考
- 🗺️ ProtoMaps 离线地图 + CyberChef 数据工具
- 💻 一键安装脚本，支持 Debian/Ubuntu

### 4. GitNexus - 代码知识图谱引擎
**链接**: https://github.com/abhigyanpatwari/GitNexus  
**核心用途**: 客户端代码知识图谱创建器，完全在浏览器中运行。输入 GitHub 仓库或 ZIP 文件，生成交互式知识图谱和内置 Graph RAG Agent。  
**为何值得关注**:
- 🔍 追踪每个依赖、调用链、集群和执行流
- 🤖 通过 MCP 连接 AI 代理 (Cursor, Claude Code, Windsurf)
- 📊 Web UI 可视化 + CLI 本地索引双模式
- 🧠 比 DeepWiki 更深度的代码分析能力
- 🔒 完全本地/浏览器内运行，隐私安全

## 📰 GitHub 平台动态

### Copilot 能力更新
- **Copilot Auto Model Selection 在 JetBrains IDEs 中 GA**  
  自动选择最佳模型（GPT-5.4, GPT-5.3-Codex, Sonnet 4.6, Haiku 4.5），付费用户享受 10% 折扣。

- **从 GitHub CLI 请求 Copilot 代码审查**  
  `gh pr create --add-reviewer @copilot` 直接在终端请求 AI 审查。

- **JetBrains IDEs 代理能力大幅增强**  
  自定义代理、子代理、计划代理现已 GA；代理钩子进入预览；MCP 自动批准支持。

- **Copilot Coding Agent Actions 工作流可跳过审批**  
  仓库管理员可设置自动运行工作流，加速反馈循环。

### 平台功能更新
- **REST API version 2026-03-10 发布**  
  首个包含破坏性变更的日历版本，2022-11-28 将继续支持至少 24 个月。

- **Issue Fields 公共预览版**  
  结构化 Issue 元数据，支持单选/文本/数字/日期类型，组织最多 25 个字段。

- **Actions OIDC Tokens 支持仓库自定义属性**  
  可将自定义属性作为声明包含在 OIDC 令牌中，简化云策略配置。
