## 🔥 今日热榜项目

### 1. Blaizzy/mlx-vlm
**链接**: <https://github.com/Blaizzy/mlx-vlm>
**Stars**: 持续增长 | **语言**: Python

**核心用途**: 基于 Apple MLX 框架的视觉语言模型（VLM）推理与微调工具包，支持图像、音频、视频多模态输入。

**为何值得关注**:
- 新增 Omni 模型支持（音频 + 视频多模态）
- 支持最新模型：Gemma 4、DeepSeek-OCR 系列、Falcon-OCR、Granite 4.0 Vision 等
- 内置 TurboQuant KV Cache，3.5-bit 量化可显著降低长上下文内存占用
- Vision Feature Cache 功能在多轮对话中实现 **11x+ 提速**
- 提供 OpenAI 兼容的 FastAPI Server（`/v1/chat/completions`）
- 支持 Thinking Budget 控制（适配 Qwen3.5 等思维链模型）

### 2. microsoft/agent-framework
**链接**: <https://github.com/microsoft/agent-framework>
**语言**: Python / .NET

**核心用途**: 微软官方多语言 AI Agent 框架，支持从简单聊天 Agent 到复杂多 Agent 图编排工作流的构建、编排和部署。

**为何值得关注**:
- 同时支持 **Python 和 .NET (C#)**，API 一致
- 图编排工作流：支持流式、检查点、人机协同、时间旅行
- DevUI：交互式开发调试界面
- AF Labs 实验性包：基准测试、强化学习等前沿功能
- 提供 **Semantic Kernel → Agent Framework** 和 **AutoGen → Agent Framework** 的迁移指南
- 内置 OpenTelemetry 可观测性支持

### 3. onyx-dot-app/onyx
**链接**: <https://github.com/onyx-dot-app/onyx>
**语言**: Python/TypeScript

**核心用途**: 开源 AI 平台，提供 RAG、Deep Research、自定义 Agent、Web 搜索、代码执行、文档生成等企业级 AI 能力。

**为何值得关注**:
- Deep Research 功能在 2026 年 2 月 benchmark 排行榜登顶
- 50+ 数据连接器（Google、Slack、Confluence 等）+ MCP 支持
- 支持所有主流 LLM 提供商（OpenAI、Anthropic、Gemini、Ollama 等）
- Lite 模式仅占 1GB 内存，适合快速试用
- 企业级特性：SSO、RBAC、SCIM、白标、审计日志

### 4. Yeachan-Heo/oh-my-codex
**链接**: <https://github.com/Yeachan-Heo/oh-my-codex>
**语言**: Node.js/TypeScript

**核心用途**: OpenAI Codex CLI 的工作流增强层，在 Codex 之上添加结构化工作流、Agent 团队协作和项目管理。

**为何值得关注**:
- 提供标准化工作流：`$deep-interview` → `$ralplan` → `$ralph`/`$team`
- `$team` 支持多 Agent 并行执行（基于 tmux）
- `.omx/` 目录持久化存储计划、日志、记忆和运行时状态
- 兼容 OpenClaw 集成指南
- 一句话安装：`npm install -g @openai/codex oh-my-codex && omx setup`
