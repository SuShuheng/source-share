## 🔥 今日爆款项目

### 1. bytedance/UI-TARS-desktop
- **链接**: https://github.com/bytedance/UI-TARS-desktop
- **核心用途**: 开源多模态 AI Agent 全栈，包含 **Agent TARS**（CLI/Web UI 通用 Agent）和 **UI-TARS Desktop**（桌面端 GUI Agent）两大产品。支持 GUI Agent、浏览器操控、MCP 工具集成。
- **为何值得关注**: 字节跳动 Agent 基础设施标杆项目，覆盖从 CLI 到桌面端的完整 GUI Agent 生态。Agent TARS 支持 hybrid browser agent（视觉+DOM 混合策略）、Event Stream 驱动的 Context Engineering。是当前最活跃的开源多模态 Agent 项目之一。

### 2. anthropics/financial-services
- **链接**: https://github.com/anthropics/financial-services
- **核心用途**: Anthropic 官方发布的金融服务垂直插件集，涵盖投资银行、股票研究、私募股权和财富管理四大领域。每个插件同时作为 **Cowork 插件** 和 **Claude Managed Agent 模板** 提供。
- **为何值得关注**: Anthropic 首次推出行业垂直 Agent 插件，标志着 Claude 从通用助手向行业专家转型的重要一步。对金融科技从业者极具参考价值。

### 3. lsdefine/GenericAgent
- **链接**: https://github.com/lsdefine/GenericAgent
- **核心用途**: 自进化 Agent——从 3300 行种子代码开始，自动生长技能树，实现完整系统控制，token 消耗降低 6 倍。Python 实现，MIT 许可。
- **为何值得关注**: ⭐ 9.2k stars，"自进化"概念极具前沿性。与传统的预定义工具链不同，GenericAgent 通过实际使用自主扩展能力边界，代表了 Agent 架构的新范式。

### 4. addyosmani/agent-skills
- **链接**: https://github.com/addyosmani/agent-skills
- **核心用途**: 面向 AI 编码 Agent 的生产级工程技能集合，包含代码审查、质量保证等技能模块。Addy Osmani（Google Chrome 团队）出品。
- **为何值得关注**: 为 Claude Code / Cursor 等 AI 编码工具提供标准化、可复用的技能定义。代码审查技能（SKILL.md）可直接集成到 Agent 工作流中，提升 AI 辅助开发的专业度。

### 5. HKUDS/AI-Trader
- **链接**: https://github.com/HKUDS/AI-Trader
- **核心用途**: 100% 全自动 Agent 原生交易系统。香港大学数据科学实验室出品。
- **为何值得关注**: 将 AI Agent 架构应用于量化交易的完整实现，从信号分析到执行全链路自动化。对 AI+金融交叉领域的研究者和从业者有重要参考意义。

## 🌟 高热度项目

### 6. affaan-m/everything-claude-code
- **链接**: https://github.com/affaan-m/everything-claude-code
- **核心用途**: Claude Code 全方位性能优化系统，包含 300+ agents、skills、commands 和 hooks，支持多 Agent 协同编码工作流、TDD、安全审计、跨语言（JS/TS/Python/Rust/Mobile）生产级代码生成与部署自动化。
- **为何值得关注**: Claude Code 生态中最全面的增强工具集，将 Claude Code 从单一编码助手升级为多 Agent 自主编码平台。

### 7. jundot/omlx
- **链接**: https://github.com/jundot/omlx
- **核心用途**: 专为 Apple Silicon 优化的开源 LLM 推理服务器，支持 continuous batching、SSD 缓存、macOS 菜单栏管理。最新版 v0.3.8，内置 Qwen 3.5/6、Gemma 4 等预设。
- **为何值得关注**: ⭐ 13k+ stars。填补了 macOS 本地 LLM 部署的空白，SSD 缓存技术让大模型在消费级 Mac 上流畅运行。v0.3.8 新增 Qwen 3.5/6 预设支持。

### 8. datawhalechina/easy-vibe
- **链接**: https://github.com/datawhalechina/easy-vibe
- **核心用途**: Datawhale 出品的 Vibe Coding 2026 入门课程，从零开始教授现代 AI 编程工具使用。高级篇涵盖 Claude Code MCP、Skills、Agent Teams 等进阶内容，附带 8 个跨平台实战项目。
- **为何值得关注**: 国内最活跃的 AI 开源社区 Datawhale 推出的系统化 Vibe Coding 教程，适合零基础入门 AI 辅助编程。

### 9. datawhalechina/hello-agents
- **链接**: https://github.com/datawhalechina/hello-agents
- **核心用途**: Datawhale Agent 教程仓库，系统化讲解 AI Agent 开发入门。
- **为何值得关注**: 与 easy-vibe 互补，聚焦 Agent 开发而非 Vibe Coding，适合想深入理解 Agent 架构的学习者。

## 🎨 非 AI 但值得关注

### 10. playcanvas/supersplat
- **链接**: https://github.com/playcanvas/supersplat
- **核心用途**: 浏览器端 3D Gaussian Splat 编辑器，基于 PlayCanvas 引擎 + WebGPU 加速。最新版 v2.25.1，支持选择、变换、优化 splat 场景。
- **为何值得关注**: 3DGS（3D Gaussian Splatting）是当前 3D 视觉最热方向之一。SuperSplat 提供了零门槛的浏览器端编辑工具，推动 3DGS 从研究走向生产。
