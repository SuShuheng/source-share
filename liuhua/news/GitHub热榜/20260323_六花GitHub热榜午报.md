## 🔥 今日热榜亮点

### 1. Claude HUD - Claude Code 的实时状态 HUD 插件

**项目**: [jarrodwatts/claude-hud](https://github.com/jarrodwatts/claude-hud)

**核心用途**: 一款 Claude Code 原生插件，在终端底部实时显示上下文使用率、工具活动、运行中的 Agent 和 Todo 进度。

**为何值得关注**:
- 🎯 **原生集成**: 使用 Claude Code 的 statusline API，无需额外窗口或 tmux
- 📊 **实时洞察**: 上下文健康度（绿→黄→红渐变）、工具活动、Agent 状态、Todo 进度
- ⚡ **超快更新**: 约 300ms 刷新一次，支持 1M 上下文窗口
- 🎨 **高度可配置**: Full/Essential/Minimal 三种预设 + 自定义颜色和阈值

**安装命令**:
```bash
/plugin marketplace add jarrodwatts/claude-hud
/plugin install claude-hud
/claude-hud:setup
```

### 3. Trivy - 全能安全扫描器

**项目**: [aquasecurity/trivy](https://github.com/aquasecurity/trivy)

**核心用途**: 全面的安全扫描器，可在容器、Kubernetes、代码仓库、云环境中查找漏洞、错误配置、敏感信息和 SBOM。

**为何值得关注**:
- 🔍 **多目标扫描**: 容器镜像、文件系统、Git 仓库、虚拟机镜像、Kubernetes
- 🛡️ **全面检测**: CVE 漏洞、IaC 错误配置、敏感信息/密钥、软件许可证
- 🔗 **丰富生态**: GitHub Actions、Kubernetes Operator、VS Code 插件
- 📦 **多渠道安装**: `brew install trivy` / Docker / 二进制下载

**快速使用**:
```bash
trivy image python:3.4-alpine
trivy fs --scanners vuln,secret,misconfig myproject/
trivy k8s --report summary cluster
```
