## 🔥 今日爆火项目

### 1. NousResearch/hermes-agent
- **链接**：https://github.com/NousResearch/hermes-agent
- **核心用途**：自学习 AI Agent，内置封闭学习循环——从经验中创建技能、使用中改进技能、自主持久化知识、跨会话搜索历史对话、构建用户画像。
- **为何值得关注**：
  - Nous Research 出品，当前热榜排名第一
  - 支持 300+ 模型（OpenRouter、Nous Portal、NVIDIA NIM、GLM、Kimi、MiniMax 等），一键切换
  - 全平台消息网关：Telegram、Discord、Slack、WhatsApp、Signal
  - 六种终端后端（本地、Docker、SSH、Singularity、Modal、Daytona），支持 Serverless 持久化
  - 兼容 agentskills.io 开放标准，支持 OpenClaw 迁移
  - MIT 开源许可

### 2. NVIDIA/cosmos
- **链接**：https://github.com/NVIDIA/cosmos
- **核心用途**：NVIDIA 开放世界模型平台，包含世界模型、数据集和工具，用于构建机器人物理 AI、自动驾驶汽车、智能基础设施等。
- **为何值得关注**：
  - NVIDIA 官方出品，Physical AI 领域重大开源项目
  - 涵盖世界基础模型（World Foundation Models），支持视频生成和物理仿真
  - 对机器人、自动驾驶、工业 AI 有直接应用价值

### 3. CopilotKit/CopilotKit
- **链接**：https://github.com/CopilotKit/CopilotKit
- **核心用途**：Agent 的前端技术栈，支持 React + Angular。AG-UI Protocol 的创造者。
- **为何值得关注**：
  - 为 AI Agent 提供完整的 Generative UI 解决方案
  - AG-UI 协议正在成为 Agent 与前端交互的标准之一
  - 社区活跃，持续迭代

### 4. chopratejas/headroom
- **链接**：https://github.com/chopratejas/headroom
- **核心用途**：压缩工具输出、日志、文件和 RAG 数据块，使其在发送给 LLM 前减少 60-95% 的 token 消耗，同时保持回答质量不变。提供库、代理和 MCP Server 三种使用方式。
- **为何值得关注**：
  - 直接解决 Agent 成本核心痛点——token 消耗
  - 支持 MCP Server 集成，适配现代 AI 工具链
  - 简洁实用，适用场景广泛

### 5. withastro/flue
- **链接**：https://github.com/withastro/flue
- **核心用途**：沙箱化 Agent 框架（实验性项目），由 Astro 团队出品。
- **为何值得关注**：
  - Astro 作者参与，框架设计质量有保障
  - 沙箱化 Agent 执行环境，安全性高
  - 实验阶段但概念新颖，值得关注发展方向

## 🤖 AI 工具与框架

### 6. affaan-m/ECC
- **链接**：https://github.com/affaan-m/ECC
- **核心用途**：Agent Harness 性能优化系统，为 Claude Code、Codex、Opencode、Cursor 等提供技能、直觉、记忆、安全和研究优先开发能力。
- **为何值得关注**：AI 编程工具的"元优化"——让 Agent 用得更好、更快、更安全。

### 7. Panniantong/Agent-Reach
- **链接**：https://github.com/Panniantong/Agent-Reach
- **核心用途**：让 AI Agent "看"到整个互联网。一个 CLI 工具读取和搜索 Twitter、Reddit、YouTube、GitHub、Bilibili、小红书——零 API 费用。
- **为何值得关注**：
  - 无需 API Key，免费抓取多个平台数据
  - 支持 Bilibili 和小红书等中文平台
  - Agent 信息获取能力的低成本解决方案

### 8. MemPalace/mempalace
- **链接**：https://github.com/MemPalace/mempalace
- **核心用途**：开源 AI 记忆系统，据称为同类最佳基准成绩，免费使用。
- **为何值得关注**：AI Agent 的长期记忆是当前热门课题，任何高性能开源方案都值得关注。

### 9. lfnovo/open-notebook
- **链接**：https://github.com/lfnovo/open-notebook
- **核心用途**：Google NotebookLM 的开源实现，更灵活、功能更丰富。
- **为何值得关注**：NotebookLM 的开源替代品，可私有化部署，适合对数据隐私有要求的场景。

### 10. mvanhorn/last30days-skill
- **链接**：https://github.com/mvanhorn/last30days-skill
- **核心用途**：AI Agent 技能模块，跨 Reddit、X、YouTube、HN、Polymarket 和全网研究任意话题，然后综合生成有据可依的摘要。
- **为何值得关注**：Agent 研究技能的标准化模块，可直接集成到 Agent 工作流。
