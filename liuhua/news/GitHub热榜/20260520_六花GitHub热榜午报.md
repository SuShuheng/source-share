## 🔥 今日爆款 TOP 10

### 1. 🏆 tinyhumansai/openhuman — 个人 AI 超级智能助理
- **链接**: https://github.com/tinyhumansai/openhuman
- **核心用途**: 开源桌面端 Agent 助手，118+ 第三方集成（Gmail/Notion/Slack/GitHub 等），一键 OAuth 连接，20 分钟自动同步数据到本地 Memory Tree + Obsidian Wiki。内置 TokenJuice 智能压缩，降低 80% token 消耗。
- **为何值得关注**: 登顶今日 GitHub Trending。将 Karpathy 的 LLM Wiki 理念产品化，实现了从安装到可用只需几分钟的体验。支持语音（STT + ElevenLabs TTS）、桌面吉祥物、Google Meet 实时参与等独特功能。GNU 开源协议。

### 2. HKUDS/CLI-Anything — 让所有软件变为 Agent 原生
- **链接**: https://github.com/HKUDS/CLI-Anything
- **核心用途**: 一行命令将任何桌面/Web 软件包装为 Agent 可用的 CLI 工具。支持 Pi、OpenClaw、Cursor、Claude Code 等 Agent 框架。配套 CLI-Hub 包管理器，社区已贡献 Obsidian、QGIS、Safari、Kdenlive、n8n 等数十个 CLI 适配器。
- **今日数据**: ⭐ 37,549 | Fork 3,596 | 今日 +1,027 ⭐
- **为何值得关注**: 解决了 Agent 生态的关键瓶颈——如何让 AI 操作现有软件。CLI-Hub 社区模式让适配器持续增长。

### 3. Imbad0202/academic-research-skills — Claude Code 学术研究技能包
- **链接**: https://github.com/Imbad0202/academic-research-skills
- **核心用途**: 为 Claude Code 提供完整的学术研究工作流：研究 → 撰写 → 审稿 → 修改 → 定稿。Python 实现。
- **今日数据**: ⭐ 13,883 | Fork 1,312 | 今日 +3,184 ⭐（今日涨幅最高！）
- **为何值得关注**: 日增 3,184 Star 是今天最高涨幅。直接面向学术写作场景，对研究人员极具实用价值。

### 4. rohitg00/agentmemory — AI 编码 Agent 持久记忆引擎
- **链接**: https://github.com/rohitg00/agentmemory
- **核心用途**: 为 Claude Code、Cursor、Codex CLI、OpenClaw、Gemini CLI 等主流编码 Agent 提供持久化记忆。基于 BM25 + Vector + Graph 混合搜索（RRF 融合），LongMemEval 基准 R@5 达 95.2%。零外部依赖，SQLite 本地存储。
- **今日数据**: ⭐ 13,901 | Fork 1,161 | 今日 +1,626 ⭐
- **为何值得关注**: 解决了编码 Agent"每次重新开始"的痛点。跨 Agent 共享同一记忆服务器，一次学习、所有 Agent 受益。年成本仅 ~$10。

### 5. colbymchenry/codegraph — 预索引代码知识图谱
- **链接**: https://github.com/colbymchenry/codegraph
- **核心用途**: 为 Claude Code、Codex、Cursor、OpenCode 预建代码知识图谱，减少 94% 工具调用和 77% 探索时间。支持 19+ 语言，100% 本地运行，FTS5 全文搜索 + 文件监听自动同步。
- **今日数据**: ⭐ 6,159 | Fork 414 | 今日 +1,869 ⭐
- **为何值得关注**: Agent 探索代码库时不再需要反复 grep/read，一次查询即可获取符号关系、调用链、代码结构。实测在 VS Code 代码库上将探索从 52 次调用降至 3 次。

### 6. CloakHQ/CloakBrowser — 隐身 Chromium 浏览器
- **链接**: https://github.com/CloakHQ/CloakBrowser
- **核心用途**: 源码级 C++ 指纹修改的 Chromium，通过 30+ 反检测测试。Playwright/Puppeteer 直接替换，3 行代码即可绕过 Cloudflare/FingerprintJS 等检测。支持 humanize 人类行为模拟。
- **今日数据**: ⭐ 16,295 | Fork 1,261 | 今日 +1,466 ⭐
- **为何值得关注**: 非传统 AI 项目但热度和价值极高。49 个 C++ 源码级指纹补丁 + reCAPTCHA v3 0.9 评分，是自动化测试/爬虫领域的重大突破。

### 7. rtk-ai/rtk — LLM Token 消耗压缩代理
- **链接**: https://github.com/rtk-ai/rtk
- **核心用途**: CLI 代理，过滤和压缩命令输出后再送入 LLM 上下文，减少 60-90% token 消耗。单个 Rust 二进制文件，零依赖，支持 100+ 命令，开销 <10ms。支持 Claude Code、Copilot、Gemini CLI、Codex、Cursor 等主流 Agent。
- **为何值得关注**: 直接降低 Agent 使用成本。示例：`git log` 从 2,500 token 压缩至 500，`pytest` 输出从 8,000 压缩至 800。

### 8. obra/superpowers — Agent 驱动软件开发方法论

- **为何值得关注**: 已入选 Anthropic 官方插件市场。不只是工具，而是一套让 Agent 按工程最佳实践工作的方法论，强调 TDD/YAGNI/DRY。
