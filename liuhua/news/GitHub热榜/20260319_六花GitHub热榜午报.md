## 🔥 爆火 AI 项目

### 1. GitNexus - 零服务器代码智能引擎
**仓库**: [abhigyanpatwari/GitNexus](https://github.com/abhigyanpatwari/GitNexus)  
**核心用途**: 客户端知识图谱创建器，在浏览器中运行，支持 Graph RAG Agent  
**为何值得关注**:
- 🧠 **知识图谱引擎**: 将任何代码库索引为知识图谱，追踪每个依赖、调用链、集群和执行流程
- 🔌 **MCP 支持**: 提供 7 个 MCP 工具，支持 Claude Code、Cursor、Windsurf 等主流 AI 编程工具
- 🌐 **零服务器架构**: 完全在浏览器中运行，无需上传代码到服务器
- 📊 **13 种语言支持**: TypeScript、JavaScript、Python、Java、Kotlin、C#、Go、Rust、PHP、Ruby、Swift、C、C++
- 🎯 **核心创新**: 预计算关系智能，让小模型也能获得完整的架构视野

**快速开始**:
```bash
npm install -g gitnexus
npx gitnexus analyze  # 一键索引代码库
```

### 2. Claude HUD - Claude Code 可视化插件
**仓库**: [jarrodwatts/claude-hud](https://github.com/jarrodwatts/claude-hud)  
**核心用途**: 显示 Claude Code 的上下文使用、活动工具、运行代理和待办进度  
**为何值得关注**:
- 👁️ **实时监控**: 显示上下文窗口使用率、活动工具、运行中的代理
- 📊 **可视化进度**: 上下文使用率可视化（绿 → 黄 → 红）
- 🔧 **高度可配置**: 支持多种预设（Full/Essential/Minimal）和自定义配置
- ⚡ **原生集成**: 使用 Claude Code 的原生 statusline API，无需额外窗口
- 🎨 **实时更新**: 每 300ms 更新一次，支持 1M 上下文窗口

**安装方式**:
```bash
/plugin marketplace add jarrodwatts/claude-hud
/plugin install claude-hud
/claude-hud:setup
```
