## 🔥 今日爆火项目

### 1. **langchain-ai/open-swe** - 开源异步编码代理框架
**项目链接**: https://github.com/langchain-ai/open-swe  
**核心用途**: 企业级内部编码代理的开源实现框架

**为何值得关注**:
- **对标顶级工程团队**: 复刻 Stripe、Ramp、Coinbase 等顶级公司的内部编码代理架构
- **企业级设计**: 支持云沙箱、Slack/Linear 集成、子代理编排、自动 PR 创建
- **技术架构**: 基于 LangGraph 和 Deep Agents，支持 Modal、Daytona、Runloop 等多种沙箱提供商
- **工具精简**: 约 15 个精选工具（执行、HTTP 请求、Git 提交、Linear 评论、Slack 回复等）
- **上下文工程**: 支持 AGENTS.md 配置文件，注入项目约定和架构决策
- **实际应用**: 从 Linear issue、Slack 线程或 GitHub PR 评论触发，自动完成任务并创建 PR

**热度依据**: GitHub Trending Daily 热榜第一名，LangChain 官方发布，提供完整的企业级编码代理解决方案

### 2. **jarrodwatts/claude-hud** - Claude Code 实时状态插件
**项目链接**: https://github.com/jarrodwatts/claude-hud  
**核心用途**: Claude Code 插件，实时显示上下文使用、工具活动、代理状态和 Todo 进度

**为何值得关注**:
- **实时监控**: 上下文健康度（精确到百分比）、工具活动、子代理运行状态、Todo 完成进度
- **原生集成**: 使用 Claude Code 的 statusline API，无需单独窗口或 tmux
- **可视化**: 绿-黄-红色彩渐变显示上下文使用情况，支持多种配置预设
- **使用限制**: 支持 Pro/Max/Team 订阅的使用率显示（1天/7天限制）
- **配置灵活**: 3 种预设（Full/Essential/Minimal），支持手动配置颜色、阈值、显示层级
- **实时更新**: 每 300ms 更新一次，支持 1M 上下文会话

**热度依据**: GitHub Trending Daily 热榜，解决 Claude Code 用户的痛点需求，提升开发体验
