### 5. Page Agent - 阿里开源的页面内 GUI Agent

**项目信息**
- **仓库**：[alibaba/page-agent](https://github.com/alibaba/page-agent)
- **Stars**：2,542 ⭐ (+465 today)
- **Forks**：199
- **语言**：TypeScript

**核心用途**
Page Agent 是一个生活在你的网页中的 GUI Agent。用自然语言控制 Web 界面。它是一个纯 JavaScript 库，所有操作都在你的网页中发生，无需浏览器扩展、Python 或无头浏览器。

**为何值得关注**
- 🎯 **轻量集成**：无需浏览器扩展/Python/无头浏览器，只需要页面内 JavaScript
- 📖 **文本驱动**：基于文本的 DOM 操作，无需截图、OCR 或多模态 LLM
- 🧠 **自带模型**：支持你自己的 LLM（qwen、gpt 等）
- 🎨 **精美 UI**：带有人类参与循环的漂亮 UI
- 🐙 **可选扩展**：可选的 Chrome 扩展，用于多页面任务

**应用场景**
- **SaaS AI Copilot**：在你的产品中快速部署 AI copilot
- **智能表单填充**：将 20 次点击的工作流转换成一句话
- **无障碍访问**：通过自然语言让任何 Web 应用可访问

**快速上手**
```html
<script src="https://cdn.jsdelivr.net/npm/page-agent@1.5.4/dist/iife/page-agent.demo.js" crossorigin="true"></script>
```

```javascript
import { PageAgent } from 'page-agent'

const agent = new PageAgent({
  model: 'qwen3.5-plus',
  baseURL: 'https://dashscope.aliyuncs.com/compatible-mode/v1',
  apiKey: 'YOUR_API_KEY',
  language: 'en-US',
})

await agent.execute('Click the login button')
```
