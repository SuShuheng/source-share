## 📊 热榜精选

### 1. 🥇 mattpocock/skills — AI 编程 Agent 技能框架
- **链接**: https://github.com/mattpocock/skills
- **今日新增**: ⭐ 7,356 | 总星: 42,826
- **语言**: Shell
- **核心用途**: 面向真实工程师的 AI 编程 Agent 技能合集。提供 `/grill-me`（需求对齐）、`/grill-with-docs`（领域语言构建）、`/tdd`（测试驱动开发）、`/diagnose`（调试循环）等模块化技能，支持 Claude Code、Codex 等主流 Agent。
- **为何值得关注**: Matt Pocock（Total TypeScript 作者）从数十年工程经验中提炼的 Agent 协作方法论。不是又一个 Agent 框架，而是教你如何用工程思维驾驭 AI 编程。`npx skills@latest add mattpocock/skills` 一键安装。

### 2. 🥈 warpdotdev/warp — Agentic 开发环境终端
- **链接**: https://github.com/warpdotdev/warp
- **今日新增**: ⭐ 11,955 | 总星: 41,944
- **语言**: Rust
- **核心用途**: 从终端诞生的 Agentic 开发环境。内置编码 Agent（Oz），也支持接入 Claude Code、Codex、Gemini CLI 等外部 Agent。支持 Agent 自动分派 Issue、编写规范、实现代码、审查 PR 的完整工作流。
- **为何值得关注**: OpenAI 作为创始赞助商支持，Rust 编写，极高性能。终端不再是黑框命令行，而是 Agent 原生的协作界面。AGPL v3 开源，MIT 双许可。

### 3. 🥉 microsoft/VibeVoice — 开源前沿语音 AI
- **链接**: https://github.com/microsoft/VibeVoice
- **今日新增**: ⭐ 1,688 | 总星: 45,538
- **语言**: Python
- **核心用途**: 微软开源的前沿语音 AI 模型家族。包含 VibeVoice-ASR（7B 参数，60 分钟长音频单次识别，支持说话人分离+时间戳）、VibeVoice-Realtime（0.5B 参数，~300ms 首次延迟的实时 TTS）。
- **为何值得关注**: ASR 支持 50+ 语言，ICLR 2026 Oral 论文。已集成 HuggingFace Transformers，支持 vLLM 推理加速。对语音识别、播客转写、多语言对话场景极具价值。

### 4. ComposioHQ/awesome-codex-skills — Codex 技能大全
- **链接**: https://github.com/ComposioHQ/awesome-codex-skills
- **今日新增**: ⭐ 1,180 | 总星: 4,644
- **语言**: Python
- **核心用途**: 为 OpenAI Codex CLI/API 策划的实用技能集合。涵盖代码迁移、CI 修复、PR 审查、Sentry 错误诊断、MCP 构建器等，支持 1000+ 应用集成（Slack、GitHub、Notion 等）。
- **为何值得关注**: "Awesome List + 可安装技能"的新范式。不只是收藏链接，而是真正可用的 Agent 能力扩展。`python skill-installer/scripts/install-skill-from-github.py` 一键安装。

### 5. abhigyanpatwari/GitNexus — 零服务端代码知识图谱
- **链接**: https://github.com/abhigyanpatwari/GitNexus
- **今日新增**: ⭐ 777 | 总星: 33,202
- **语言**: TypeScript
- **核心用途**: 将任意代码库索引为知识图谱（依赖关系、调用链、执行流），通过 MCP 接口赋能 AI Agent。浏览器端零服务器运行，也支持 CLI 模式接入 Cursor/Claude Code/Codex/Windsurf。
- **为何值得关注**: "DeepWiki 的深层版本"——不只是理解代码，而是分析代码架构。内置 Graph RAG Agent，让小模型也能获得完整架构视图。企业版支持 PR 影响范围分析。

### 6. obra/superpowers — Agentic 软件开发方法论
- **链接**: https://github.com/obra/superpowers
- **今日新增**: ~432 | 总星: 5,264
- **核心用途**: 完整的 Agent 驱动软件开发方法论。从需求脑暴→设计文档→实现计划→子 Agent 并行开发→TDD→代码审查→分支合并，全流程自动化。支持 Claude Code、Codex、Cursor、OpenCode、GitHub Copilot、Gemini。
- **为何值得关注**: 不是单个工具，而是将工程最佳实践（YAGNI、DRY、红绿重构）系统化注入 Agent 工作流。Jesse Vincent（RT 创始人）出品，已在 Claude 官方插件市场上架。

### 7. 1jehuang/jcode — 极致性能编码 Agent 终端
- **链接**: https://github.com/1jehuang/jcode
- **今日新增**: ⭐ 386 | 总星: 1,203
- **语言**: Rust
- **核心用途**: 下一代编码 Agent Harness，主打多会话工作流和极致性能。内置语义记忆系统（向量嵌入 + 图谱检索），支持自动记忆提取和整合。单会话仅 27.8 MB RAM，启动 14ms。
- **为何值得关注**: 性能数据碾压同类——比 Claude Code 快 245.5x 启动，10 会话仅用 Claude Code 1/9 的内存。自研 mermaid-rs-renderer 渲染速度提升 1800x。适合重度 Agent 用户。

### 8. CJackHwang/ds2api — DeepSeek Web 转 API 中间件
- **链接**: https://github.com/CJackHwang/ds2api
- **今日新增**: ⭐ 461 | 总星: 2,653
- **语言**: Go
- **核心用途**: 轻量高性能全栈中间件，将 DeepSeek Web 对话能力转换为兼容 OpenAI / Claude / Gemini API 格式。支持多账号轮询、Docker / Vercel 部署。
- **为何值得关注**: Go 全量实现，React WebUI 管理台。中文社区热门项目，解决了 DeepSeek API 接入的实际痛点。需注意仅供学习研究使用。
