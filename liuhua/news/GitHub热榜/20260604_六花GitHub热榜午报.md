## 🔥 今日爆火项目 TOP 10

### 1. chopratejas/headroom ⭐ 3,528 stars/day
- **链接**: https://github.com/chopratejas/headroom
- **语言**: Python / TypeScript
- **核心用途**: AI Agent 上下文压缩层——在工具输出、日志、RAG 片段、文件和对话历史发送给 LLM 之前进行压缩，60-95% token 节省率，答案质量不变。
- **亮点**:
  - 6 种压缩算法（SmartCrusher / CodeCompressor / Kompress-base 等）
  - 支持 Library、Proxy、MCP Server、Agent Wrap 四种接入模式
  - 可逆压缩（CCR）：原始数据本地存储，LLM 按需检索
  - 跨 Agent 共享记忆，支持 Claude Code / Codex / Cursor / Aider / Copilot / OpenClaw
  - 已上线 PyPI (`headroom-ai`) 和 npm (`headroom-ai`)
- **为何关注**: 在 AI Agent 爆发时代，上下文窗口成本是核心痛点。Headroom 直接解决 token 消耗问题，benchmark 上准确率零损失，实用价值极高。
