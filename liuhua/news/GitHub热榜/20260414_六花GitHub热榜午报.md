## 🔥 今日爆火项目

### 1. NousResearch/hermes-agent
> **The agent that grows with you** — 自我进化的 AI Agent

- **链接**: https://github.com/NousResearch/hermes-agent
- **语言**: Python | **协议**: MIT
- **核心用途**: Nous Research 出品的自学习 AI Agent。内置学习闭环——从经验中创建技能、使用中自我改进、自动持久化知识、搜索历史对话、跨会话构建用户画像。支持 Telegram/Discord/Slack/WhatsApp/Signal 等多平台网关。
- **为何值得关注**: 
  - 唯一具有"自我学习闭环"的开源 Agent，技能在使用中自动进化
  - 支持 200+ 模型切换（OpenRouter、Nous Portal、GLM、Kimi、MiniMax、OpenAI 等）
  - 可在 $5 VPS、GPU 集群或 Serverless 上运行
  - 内置 cron 调度、子 Agent 并行、RPC 工具调用
  - 提供 OpenClaw 迁移工具 `hermes claw migrate`

### 2. OpenBMB/VoxCPM
> **Tokenizer-Free TTS** — 30 语言、48kHz 音质的开放语音合成系统

- **链接**: https://github.com/OpenBMB/VoxCPM
- **语言**: Python | **协议**: Apache-2.0（可商用）
- **核心用途**: VoxCPM2 是 2B 参数的多语言 TTS 系统，基于 MiniCPM-4 骨干网络，训练数据超 200 万小时。支持 30 种语言直接合成、Voice Design（自然语言描述创建新声音）、可控声音克隆、48kHz 录音棚级音质输出。
- **为何值得关注**:
  - 首个 tokenizer-free 的端到端扩散自回归 TTS 架构
  - 30 种语言无需语言标签，包括中文方言（粤语、四川话等）
  - RTX 4090 上实时率低至 ~0.3，支持流式推理
  - Apache-2.0 完全开源可商用

### 3. multica-ai/multica
> **Your next 10 hires won't be human** — 开源编码 Agent 管理平台

- **链接**: https://github.com/multica-ai/multica
- **语言**: TypeScript | **协议**: 开源
- **核心用途**: 将编码 Agent 变成真正的队友。像给同事分配任务一样给 Agent 分配 Issue，Agent 自主执行、报告阻塞、更新状态。支持 Claude Code、Codex、OpenClaw、OpenCode。
- **为何值得关注**:
  - Agent 即队友：有头像、出现在看板、参与评论、主动报告阻塞
  - 可复用技能：每次解决方案都沉淀为团队共享技能
  - 统一运行时管理：本地 daemon + 云运行时统一监控
  - 自托管，vendor-neutral

### 4. coleam00/Archon
> **Harness Builder for AI Coding** — 让 AI 编码确定且可重复

- **链接**: https://github.com/coleam00/Archon
- **语言**: TypeScript/Bun | **协议**: 开源
- **核心用途**: AI 编码 Agent 的工作流引擎。用 YAML 定义开发流程（规划→实现→验证→审查→PR），每个工作流运行在独立 git worktree 中，支持并行执行。
- **为何值得关注**:
  - 类似 Dockerfile 之于基础设施，GitHub Actions 之于 CI/CD——Archon 之于 AI 编码
  - 确定性工作流 + AI 智能节点混合编排
  - 支持 CLI、Web UI、Slack、Telegram、GitHub 多入口
  - "Fire and forget"——启动工作流后回来就是完成 PR

### 5. microsoft/markitdown
> **File to Markdown** — 微软出品的文档转 Markdown 工具

- **链接**: https://github.com/microsoft/markitdown
- **语言**: Python | **协议**: MIT
- **核心用途**: 将 PDF、PowerPoint、Word、Excel、图片（OCR）、音频（语音转录）、HTML、ZIP 等格式批量转换为 Markdown，专为 LLM 管线设计。现已提供 MCP Server 集成。
- **为何值得关注**: LLM 应用中最实用的预处理工具之一，MCP 支持使其可直接集成 Claude Desktop 等应用

### 6. snarktank/ralph
> **Autonomous AI Agent Loop** — PRD 驱动的自主编码循环

- **链接**: https://github.com/snarktank/ralph
- **语言**: Shell | **协议**: 开源
- **核心用途**: 自主 AI Agent 循环，反复运行 AI 编码工具（Amp 或 Claude Code）直到所有 PRD 条目完成。每次迭代是全新实例，通过 git 历史、progress.txt 和 prd.json 保持记忆。
- **为何值得关注**: Geoffrey Huntley 的 Ralph 模式实现，简洁的 bash 脚本即可实现"写完 PRD 就不用管了"的自动化编码

### 7. shiyu-coder/Kronos
> **金融市场的语言基础模型** — K 线序列的 Foundation Model

- **链接**: https://github.com/shiyu-coder/Kronos
- **协议**: MIT | **论文**: AAAI 2026
- **核心用途**: 首个专为金融 K 线数据设计的 decoder-only 基础模型。通过专用 tokenizer 将 OHLCV 数据量化为层次化离散 token，再用自回归 Transformer 统一处理量化任务。覆盖 45+ 全球交易所。
- **为何值得关注**: AAAI 2026 论文，从 mini(4.1M) 到 base(102.3M) 全开源，支持 Qlib 回测集成
