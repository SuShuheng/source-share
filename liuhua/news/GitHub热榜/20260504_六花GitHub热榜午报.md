## 🔥 今日热榜精选

### 1. TauricResearch/TradingAgents ⭐ 热度最高
- **链接**: https://github.com/TauricResearch/TradingAgents
- **语言**: Python
- **核心用途**: 多 Agent LLM 金融交易框架。模拟真实交易公司的运作模式，部署基本面分析师、情绪分析师、新闻分析师、技术分析师、研究员团队、交易员、风险管理团队和投资组合经理等多角色 LLM Agent，通过结构化辩论和多轮协作做出交易决策。
- **为何值得关注**:
  - 架构设计精良：LangGraph 构建，支持 OpenAI / Google / Anthropic / xAI / DeepSeek / Qwen / GLM / Ollama 等十余种 LLM Provider
  - 最新 v0.2.4 新增结构化输出 Agent、检查点恢复、持久化决策日志
  - 有学术论文支撑 (arXiv:2412.20138)，社区活跃
  - 金融 AI + Multi-Agent 是当前热门交叉领域

### 2. ruvnet/ruflo（原 Claude Flow）
- **链接**: https://github.com/ruvnet/ruflo
- **语言**: TypeScript / Rust (WASM)
- **核心用途**: Claude Code 的多 Agent 编排平台。支持 100+ 专业 Agent 的集群调度、自我学习记忆系统、联邦通信和企业级安全。WASM 内核用 Rust 编写，驱动策略引擎、嵌入和证明系统。
- **为何值得关注**:
  - 从 Claude Flow 更名升级，定位为 Claude Code 的"神经系统"
  - 支持 32 个插件模块（swarm、autopilot、federation、workflows 等）
  - 自学习 Agent 架构：Agent 可从每次任务中学习并跨会话记忆
  - MCP + CLI 双模式接入

### 3. 1jehuang/jcode
- **链接**: https://github.com/1jehuang/jcode
- **语言**: Rust / TypeScript
- **核心用途**: 新一代 Coding Agent Harness，面向多会话工作流设计，强调极致性能和无限可定制性。
- **为何值得关注**:
  - **性能碾压**：单会话仅 27.8MB 内存（关闭本地嵌入），Claude Code 是 386.6MB（13.9 倍）；10 会话仅 117MB vs Claude Code 2.3GB（19.7 倍）
  - 启动速度：14ms 首帧，Claude Code 需 3.4 秒（245 倍差距）
  - 超越插件的可定制性架构，Agent 可自我配置
  - 对 AI Coding Agent 性能瓶颈问题提出了激进的解决方案

### 4. browserbase/skills
- **链接**: https://github.com/browserbase/skills
- **语言**: TypeScript
- **核心用途**: Claude Code 的浏览器自动化技能包。提供 Web 浏览器交互、反检测、CAPTCHA 解决、住宅代理、无服务器浏览器函数部署、UI 测试、Cookie 同步等完整工具链。
- **为何值得关注**:
  - 让 Claude Code 直接操控浏览器，补全了 Agent 在 Web 环境中的操作能力
  - 10 个技能模块覆盖从基础浏览到高级调试的全链路
  - Browserbase 是当前 AI Agent 基础设施赛道的活跃玩家

### 5. soxoj/maigret
- **链接**: https://github.com/soxoj/maigret
- **语言**: Python
- **核心用途**: 仅凭用户名即可从 3000+ 网站收集个人信息档案的 OSINT 工具。无需 API Key，支持递归搜索、Tor/I2P、CAPTCHA 绕过。
- **为何值得关注**:
  - 安全研究 / OSINT 领域标杆工具
  - 3000+ 站点覆盖，自动更新数据库
  - 提供 Web 界面和 Telegram Bot
  - 适用于安全审计和社会工程防护
