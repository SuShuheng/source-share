## 🔥 今日热榜 TOP 项目

### 1. thunderbird/thunderbolt ⭐ ~2,060
- **链接**: <https://github.com/thunderbird/thunderbolt>
- **语言**: TypeScript | **协议**: MPL-2.0
- **核心用途**: Thunderbird 出品的开源跨平台 AI 客户端，支持 Web/iOS/Android/Mac/Linux/Windows，可本地部署（on-prem），兼容前沿模型、本地模型和私有化模型。
- **为何值得关注**: Mozilla 体系下少见的 AI 产品，主打"你控制的 AI"——自选模型、自有数据、零供应商锁定。目前仍处于早期阶段，正在进行安全审计，面向企业客户。支持 Docker/K8s 自部署。
- **标签**: #AI客户端 #本地部署 #跨平台 #企业级

### 2. BasedHardware/omi
- **链接**: <https://github.com/BasedHardware/omi>
- **语言**: 多语言（Swift/Python/Flutter/Rust/C） | **协议**: MIT
- **核心用途**: 开源 AI 第二大脑，捕获屏幕画面和对话录音，实时转写，生成摘要和行动项，提供记忆一切的 AI 聊天。支持桌面/手机/可穿戴设备。
- **为何值得关注**: 30万+ 专业人士在用，全栈开源（从固件到 app 到后端），甚至有配套硬件（Omi Wearable、Omi Glass）。MCP Server 集成使其可与 AI Agent 生态无缝对接。
- **标签**: #AI助手 #可穿戴 #第二大脑 #全栈开源

### 3. openai/openai-agents-python ⭐ 22,975

### 4. deepseek-ai/DeepGEMM ⭐ 6,656

## 📡 GitHub 官方动态（Changelog & Engineering）

### GitHub Copilot CLI 支持自动模型选择
- **来源**: [GitHub Changelog](https://github.blog/changelog/2026-04-17-github-copilot-cli-now-supports-copilot-auto-model-selection)
- **要点**: Copilot CLI 新增 auto 模式，自动选择最高效模型（GPT-5.4、GPT-5.3-Codex、Sonnet 4.6、Haiku 4.5），享受 10% 折扣。对所有 Copilot 计划 GA。
- **影响**: 开发者无需手动选模型，Copilot 自动路由最优选项。

### GitHub CLI 新增 `gh skill` 命令管理 Agent Skills
- **来源**: [GitHub Changelog](https://github.blog/changelog/2026-04-16-manage-agent-skills-with-github-cli)
- **要点**: `gh skill` 命令支持发现、安装、管理和发布 Agent Skills，遵循开放 [Agent Skills 规范](https://agentskills.io)，跨平台兼容（Copilot、Claude Code、Cursor、Codex、Gemini CLI）。
- **影响**: Agent Skill 生态标准化的重要一步，GitHub 官方推动。

### GitHub 如何使用 eBPF 提升部署安全性
- **来源**: [GitHub Engineering Blog](https://github.blog/engineering/infrastructure/how-github-uses-ebpf-to-improve-deployment-safety/)
- **要点**: GitHub 用 eBPF 检测和防止部署脚本中的循环依赖（直接依赖、隐藏依赖、传递依赖），确保在 GitHub 自身宕机时仍可安全部署。
- **影响**: 大规模基础设施部署安全的技术分享，eBPF 在 DevOps 中的实战案例。
