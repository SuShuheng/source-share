## 🔥 今日热榜精选

### 1. Alishahryar1/free-claude-code ⭐ 爆火
- **链接**: https://github.com/Alishahryar1/free-claude-code
- **语言**: Python
- **核心用途**: 免费使用 Claude Code CLI 和 VSCode 扩展的轻量代理。将 Anthropic API 调用路由到 NVIDIA NIM（40 req/min 免费）、OpenRouter、DeepSeek、LM Studio 或 llama.cpp，无需 Anthropic API Key。
- **为何值得关注**: 零成本使用 Claude Code 的方案，支持 5 种后端提供商、按模型映射路由、thinking token 解析、智能限流、Discord/Telegram Bot 集成。drop-in 替换，只需设置 2 个环境变量。在开发社区引发巨大关注。

### 2. deepseek-ai/DeepEP
- **链接**: https://github.com/deepseek-ai/DeepEP
- **语言**: CUDA
- **核心用途**: DeepSeek 开源的高效专家并行（Expert Parallelism）通信库，专为 Mixture-of-Experts (MoE) 架构设计。提供高吞吐、低延迟的 all-to-all GPU kernel（MoE dispatch & combine）。
- **为何值得关注**: DeepSeek 持续开源 MoE 基础设施核心组件。对于大规模 MoE 模型训练和推理部署至关重要，是 AI 基础设施领域的重要工具。

### 3. zilliztech/claude-context
- **链接**: https://github.com/zilliztech/claude-context
- **语言**: TypeScript
- **核心用途**: 为 Claude Code 及其他 AI 编程代理提供语义代码搜索的 MCP 插件。将整个代码库向量化存储，按相关性精准检索代码片段注入上下文，大幅降低大代码库的 token 消耗。
- **为何值得关注**: 解决了 AI 编程代理的核心痛点——大代码库的上下文管理。支持 Claude Code、Cursor、VS Code、Gemini CLI 等多种 MCP 客户端，基于 Zilliz 向量数据库。

### 4. microsoft/typescript-go
- **链接**: https://github.com/microsoft/typescript-go
- **核心用途**: 微软将 TypeScript 编译器从 TypeScript 自身移植到 Go 语言的项目。
- **为何值得关注**: TypeScript 官方团队主导的重量级工程，旨在大幅提升 TypeScript 编译速度。Go 的原生并发和性能优势可显著改善大型项目的编译体验。这是 TypeScript 生态的里程碑事件。

### 5. huggingface/ml-intern
- **链接**: https://github.com/huggingface/ml-intern
- **核心用途**: HuggingFace 新项目（详情待公开确认，推测为机器学习实习生训练/入门框架）。
- **为何值得关注**: HuggingFace 出品即热度保证。作为 ML 社区最具影响力的组织之一，新项目值得持续跟踪。

### 6. agentscope-ai/QwenPaw (v1.1.4)
- **链接**: https://github.com/agentscope-ai/QwenPaw
- **核心用途**: 个人 AI 助手，支持本地或云端部署，集成多种聊天应用，能力可轻松扩展。最新发布 v1.1.4。
- **为何值得关注**: 基于 Qwen 模型的本地 AI 助手方案，隐私友好，适合个人和企业私有化部署。

### 7. Anil-matcha/Open-Generative-AI
- **链接**: https://github.com/Anil-matcha/Open-Generative-AI
- **核心用途**: 开源生成式 AI 项目/工具/模型的精选列表。
- **为何值得关注**: 生成式 AI 领域的 awesome-list，帮助开发者快速发现高质量开源 AI 资源。

## 🏢 非 AI 但高价值项目

### 8. google/osv-scanner
- **链接**: https://github.com/google/osv-scanner
- **核心用途**: Google 出品的开源漏洞扫描器，对照 OSV 数据库扫描项目依赖中的已知安全漏洞。
- **为何值得关注**: 供应链安全是当前软件工程的重点，Google 官方维护，CI/CD 集成友好。

### 9. dani-garcia/vaultwarden
- **链接**: https://github.com/dani-garcia/vaultwarden
- **语言**: Rust
- **核心用途**: Bitwarden 服务端的轻量替代实现，用 Rust 编写，资源占用极低。
- **为何值得关注**: 自托管密码管理方案的首选，长期霸榜 GitHub trending。

### 10. PostHog/posthog
- **链接**: https://github.com/PostHog/posthog
- **核心用途**: 开源产品分析平台，提供事件追踪、功能开关、A/B 测试、会话回放等一站式产品分析工具。
- **为何值得关注**: 替代 Amplitude/Mixpanel 的最佳开源方案，all-in-one 产品分析。
